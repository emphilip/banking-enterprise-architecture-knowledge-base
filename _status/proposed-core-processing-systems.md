# Proposed Core Processing Systems

Research-stage proposals for genuinely-missing, notable Core Processing systems NOT already in
`glossary/_canonical-names.md` (verified against both the Legacy and Modern systems tables on
2026-06-17). Each realizes one of the 12 Core Processing top-level tech capabilities. Modern entries
supersede a real, SAME-capability legacy system already in (or proposed in) the registry.

Coverage note: Core Processing is already heavily covered. Payment Orchestration, Card Processing,
Loan Origination Platform, Credit Decisioning Engine, Asset Liability Management Engine, Portfolio
Management System, and Financial Consolidation Platform were judged ADEQUATE — no clearly-missing
incumbents found worth adding, so none are proposed there. Proposals concentrate on the few notable
absences: Core Banking Processing (Avaloq, Sopra, FIS MBP), General Ledger Engine (Infor, Sage),
Order Management System (Fidessa, Linedata, Calypso, FlexTrade), Investment Advisory Platform
(Objectway, Orion), and Treasury Management System (Coupa Treasury).

Note on supersedes pairs added WITHIN this proposal (legacy proposed here + modern proposed here):
Infor SunSystems -> Sage Intacct; Avaloq Banking Suite -> (FIS MBP supersedes an existing FIS legacy
core instead); Fidessa / Linedata Longview -> FlexONE; Objectway -> Orion. Calypso is proposed as a
legacy incumbent (no modern supersede proposed; would pair with an existing modern OMS if desired).
