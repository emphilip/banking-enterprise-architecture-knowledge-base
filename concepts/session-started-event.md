# Session Started Event

**Definition.** Session Started Event is the business event raised when a customer channel session is opened. Session Started Event signals the start of the channel servicing flow and identifies the entry channel and intent.

**Also known as:** Channel Session Opened Event, Session Init Event.

## Relationships
- Session Started Event is related to the Channels & Engagement domain.
- Session Started Event causes Open Channel Session.
- Session Started Event mentions Channel Servicing.
- Session Started Event is related to Channel Session.

## Details
Session Started Event marks the moment a customer engages a channel, whether digital, IVR or branch, prompting the platform to instantiate a stateful channel session and capture context. The event drives downstream authentication, intent identification and journey orchestration, and is recorded as part of the cross-channel interaction history that lets servicing follow the customer across touchpoints.
