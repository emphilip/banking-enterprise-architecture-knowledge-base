# Branch Manager

**Definition.** Branch Manager is the branch leader accountable for branch cash controls, vault dual custody, and the end-of-day reconciliation and secure closure of the branch. Branch Manager owns the opening and closing controls of the in-branch operations flow.

**Also known as:** Branch Supervisor, Head Teller.

## Relationships
- Branch Manager is related to the Channels & Engagement domain.
- Branch Manager mentions Branch Operations.
- Branch Manager mentions Open Branch Cash.
- Branch Manager mentions Reconcile Branch Cash.

## Details
Branch Manager opens the branch by issuing teller floats from the vault under dual custody, supervises counter activity during the day, and at close consolidates teller and vault positions to reconcile the branch cash general ledger before returning cash to the vault. The role signs off opening balances, enforces drawer and vault limits, investigates variances, and finalises end-of-day controls and branch closure with vault and premises securing.
