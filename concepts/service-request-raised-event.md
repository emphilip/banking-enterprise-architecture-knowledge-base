# Service Request Raised Event

**Definition.** Service Request Raised Event is the trigger raised when a customer submits a service request, starting the Service Request Handling flow. Service Request Raised Event signals that a ticket must be captured.

**Also known as:** Request Submitted Event, Ticket Raised.

## Relationships
- Service Request Raised Event is related to the Customer Management domain.
- Service Request Raised Event causes Capture Request.
- Service Request Raised Event mentions Service Request Handling.
- Service Request Raised Event is related to Request Intake & Triage.

## Details
Service Request Raised Event is emitted when a customer asks for a change or service through any channel. It initiates the Service Request Handling flow, prompting the Service Advisor to capture the request as a ticket, authenticate the requester and triage it to the appropriate priority and queue.
