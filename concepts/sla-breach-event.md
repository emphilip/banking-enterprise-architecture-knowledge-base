# SLA Breach Event

**Definition.** SLA Breach Event is the escalation event raised when an acknowledgement, final-response or fulfilment SLA threshold is breached. SLA Breach Event triggers escalation in the Complaint Handling and Service Request Handling flows.

**Also known as:** SLA Escalation Event, Deadline Breached.

## Relationships
- SLA Breach Event is related to the Customer Management domain.
- SLA Breach Event causes Issue Final Response.
- SLA Breach Event causes Fulfil Request.
- SLA Breach Event mentions Complaint Handling.
- SLA Breach Event mentions Service Request Handling.

## Details
SLA Breach Event fires when a time-bound commitment, such as the complaint acknowledgement or final-response deadline or a service-request fulfilment target, is at risk or missed. It drives escalation and prioritisation so the Complaints Officer or Service Advisor can expedite the outstanding response and bring the case back within tolerance.
