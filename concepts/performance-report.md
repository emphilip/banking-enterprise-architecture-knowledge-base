# Performance Report

**Definition.** Performance Report is the client-facing report of time-weighted returns against benchmark, with attribution and required disclosures, produced and delivered at the end of the wealth performance flow.

**Also known as:** Investment Performance Report, Client Statement (Investments).

## Relationships
- Performance Report is related to the Wealth & Investments domain.
- Performance Report mentions Performance Reporting.
- Performance Report mentions Produce Performance Report.

## Details
Performance Report presents portfolio valuation and holdings at period end, time-weighted returns for the period and since inception, the benchmark return and excess return, and an attribution breakdown of excess return into allocation and selection effects. Performance Report also shows transactions, income, fees and cash flows for the period and carries GIPS-aligned disclosures on methodology, benchmark definition and fee treatment. Performance Report is assembled after positions are valued, corporate actions are processed, positions are reconciled to custody, returns are calculated and attribution is computed. It is signed off under four-eyes review for disclosure accuracy and delivered to the client electronically or in print.
