# Loan Application

**Definition.** Loan Application is the borrower-submitted request and financial data that initiates loan origination. Loan Application is the input artifact that opens the origination flow.

**Also known as:** Credit Application, Loan Request.

## Relationships
- Loan Application is related to the Lending & Credit domain.
- Loan Application mentions Loan Origination Workflow.
- Loan Application mentions Capture Loan Application.
- Loan Application mentions Application Origination & Pre-Qualification.

## Details
Loan Application carries the borrower's identity, contact and employment details, the requested amount, term and purpose, declared income and obligations, and consent to a credit pull. It is captured into the origination platform at the start of the Loan Origination Workflow, where KYC identity capture and data-quality validation are applied before pre-qualification. For mortgages the equivalent intake uses the URLA/Form 1003. Loan Application is enriched with verification documents into the file that the Credit Underwriter decisions.
