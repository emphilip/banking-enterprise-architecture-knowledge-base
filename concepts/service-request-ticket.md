# Service Request Ticket

**Definition.** Service Request Ticket is the ticket capturing a customer service request, its type, priority/SLA, fulfilment actions and closure. Service Request Ticket is the working artifact carried through the Service Request Handling flow.

**Also known as:** Service Ticket, Request Ticket.

## Relationships
- Service Request Ticket is related to the Customer Management domain.
- Service Request Ticket mentions Capture Request.
- Service Request Ticket mentions Request Intake & Triage.
- Service Request Ticket mentions Service Request Handling.

## Details
Service Request Ticket is created when the Service Advisor captures a request and records the request type, requester verification, assigned priority and SLA, fulfilment actions and closure confirmation. It tracks the case through triage, assessment and fulfilment, and supports the reopen window and feedback capture after closure.
