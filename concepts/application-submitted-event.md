# Application Submitted Event

**Definition.** Application Submitted Event is the trigger raised when a prospect submits an onboarding application, starting the Customer Onboarding flow. Application Submitted Event signals that an onboarding case can be opened.

**Also known as:** Onboarding Started Event, Application Received.

## Relationships
- Application Submitted Event is related to the Customer Management domain.
- Application Submitted Event causes Receive Application.
- Application Submitted Event mentions Customer Onboarding.
- Application Submitted Event is related to Application Intake & Capture.

## Details
Application Submitted Event is raised by a digital channel, branch or API when a prospect completes and submits an onboarding application. It initiates the Customer Onboarding flow, prompting the Onboarding Specialist to receive the application and open an onboarding case for consent capture, document collection and verification.
