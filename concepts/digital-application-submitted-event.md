# Digital Application Submitted Event

**Definition.** Digital Application Submitted Event is the event raised when a digital onboarding application is submitted by an applicant. Digital Application Submitted Event signals that the captured application is ready for identity verification within the digital onboarding journey.

**Also known as:** Online Application Submitted Event, DAO Submitted Event.

## Relationships
- Digital Application Submitted Event is related to the Channels & Engagement domain.
- Digital Application Submitted Event causes Verify Digital Identity.
- Digital Application Submitted Event mentions Digital Onboarding Journey.
- Digital Application Submitted Event is related to Digital Application.

## Details
Digital Application Submitted Event fires once the applicant completes and submits the online account-opening application with consents captured. The event advances the Digital Onboarding Journey from intake into electronic identity verification, where document authenticity and biometric or liveness checks are run, and it anchors application tracking so that abandonment, save-and-resume and exception handling can be measured and managed.
