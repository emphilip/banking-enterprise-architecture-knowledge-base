# Corporate Action Announced Event

**Definition.** Corporate Action Announced Event is the business event signalling that an issuer has announced a corporate action requiring entitlement calculation and, where elective, an election, before performance can be measured in the wealth flow.

**Also known as:** CA Announced Event, Issuer Event.

## Relationships
- Corporate Action Announced Event is related to the Wealth & Investments domain.
- Corporate Action Announced Event mentions Performance Reporting.
- Corporate Action Announced Event causes Process Corporate Actions.

## Details
Corporate Action Announced Event fires when an issuer or custodian announces an action such as a dividend, coupon, stock split, rights issue, merger or tender offer, carrying the security identifier, event type, key dates (ex-date, record date, pay date, election cut-off) and the terms. Corporate Action Announced Event distinguishes mandatory events, which adjust positions and cash automatically, from voluntary events, which require an election before the cut-off. Corporate Action Announced Event initiates capture of the event, calculation of entitlements and adjustment of positions and cash, so the portfolio is correctly stated before reconciliation and return measurement. Missed election cut-offs are a key operational control point.
