# Conversation Tools (INT-CVT)

**L0:** [[INT Interaction Mgmt.|Interaction Mgmt.]]

Conversation Tools provide the automation engines that conduct or assist conversations across voice and digital channels.

| INT-CVT-03 | [[INT-CVT-03 Predictive Dialer\|Predictive Dialer]] | Outbound dialing that paces calls to agent availability to maximize talk time |
