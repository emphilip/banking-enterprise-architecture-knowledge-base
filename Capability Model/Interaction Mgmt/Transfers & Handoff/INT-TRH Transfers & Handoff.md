# Transfers & Handoff (INT-TRH)

**L0:** [[INT Interaction Mgmt.|Interaction Mgmt.]]

Transfers & Handoff move an interaction between resources, channels, or automation while preserving context.

## L2 Capabilities

| ID | Capability | Summary |
|---|---|---|
| INT-TRH-01 | [[INT-TRH-01 Omni-Transfer\|Omni-Transfer]] | Transferring an interaction across channels while carrying its full context |
| INT-TRH-02 | [[INT-TRH-02 Bot-to-Agent\|Bot-to-Agent]] | Escalating an automated conversation to a live agent with history intact |
| INT-TRH-03 | [[INT-TRH-03 Warm Transfer\|Warm Transfer]] | Transfer with an introductory hand-off so the receiving agent has context |
| INT-TRH-04 | [[INT-TRH-04 Cold Transfer\|Cold Transfer]] | Direct transfer of an interaction without a preceding briefing |
