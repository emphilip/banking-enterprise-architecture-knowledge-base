# Interaction Mgmt. (INT)

**Parent:** [[00 - Master Capability Model]]

Interaction Mgmt. orchestrates how customer contacts are routed, queued, conducted, and handed off across channels. It is the real-time engagement layer between [[CHN Channels|Channels]] and the agents and tools in [[OPS Operations|Operations]], ensuring each interaction reaches the right resource with full context.
