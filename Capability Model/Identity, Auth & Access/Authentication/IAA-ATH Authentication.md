# Authentication (IAA-ATH)

**L0:** [[IAA Identity, Auth & Access|Identity, Auth & Access]]

Authentication verifies that a returning user is who they claim to be at the point of access.

## L2 Capabilities

| ID | Capability | Summary |
|---|---|---|
| IAA-ATH-01 | [[IAA-ATH-01 Password - PIN\|Password / PIN]] | Knowledge-based authentication using passwords and PINs |
| IAA-ATH-02 | [[IAA-ATH-02 Multi-Factor Auth\|Multi-Factor Auth]] | Combining two or more independent factors to authenticate a user |
| IAA-ATH-03 | [[IAA-ATH-03 Biometric\|Biometric]] | Fingerprint, face, and other biometric authentication methods |
| IAA-ATH-04 | [[IAA-ATH-04 Voice Biometrics\|Voice Biometrics]] | Authenticating callers by voiceprint in phone channels |
| IAA-ATH-05 | [[IAA-ATH-05 SSO\|SSO]] | Single sign-on across the bank's applications and properties |
| IAA-ATH-06 | [[IAA-ATH-06 Passwordless\|Passwordless]] | Authentication without passwords, using device keys, passkeys, or magic links |
