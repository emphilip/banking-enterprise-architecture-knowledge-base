# Databricks

**Definition.** Databricks is a cloud-based data and AI platform from Databricks built on the lakehouse architecture, unifying data warehousing, data engineering, and machine learning.
**Also known as:** Databricks Lakehouse.

## Relationships
- Data Warehousing depends on Databricks.
- Databricks supersedes Oracle Exadata.
- Databricks is related to Snowflake.

## Details
Databricks combines a data lake and warehouse into a single lakehouse built on Apache Spark and the Delta Lake format, delivered as a managed cloud service. It supports large-scale ETL, SQL analytics, and the full machine-learning lifecycle on shared data. This modernizes legacy data appliances by enabling elastic compute and integrated AI/ML on open data formats.
