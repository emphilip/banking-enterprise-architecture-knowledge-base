# i2c

**Definition.** i2c is a cloud-native unified issuer-processing platform supporting credit, debit and prepaid programs on a single configurable stack with building-block APIs for card programs.
**Also known as:** i2c Inc, i2c Issuer Processing.

## Relationships
- Card Processing depends on i2c.
- i2c supersedes TSYS TS2.
- i2c supersedes Fiserv Optis.
- i2c is related to Pismo.

## Details
i2c runs credit, debit and prepaid card programs on a single configurable platform, exposing building-block APIs for authorization, account management and card controls. It performs ISO 8583 authorization processing and EMV-based issuing with scheme certification across Visa and Mastercard. Its cloud-native, parameter-driven model lets issuers configure programs without the code changes required by older monolithic processors.
