# Temporal

**Definition.** Temporal is a durable workflow orchestration platform from Temporal Technologies that executes long-running, fault-tolerant workflows defined in code.
**Also known as:** Temporal.io.

## Relationships
- Workflow Orchestration depends on Temporal.
- Temporal supersedes Pega BPM.
- Temporal is related to Camunda.

## Details
Temporal lets developers write workflows as code while the platform guarantees durable execution, automatic retries, and state persistence across failures. Available self-hosted or as Temporal Cloud, it scales reliable orchestration of microservices and asynchronous processes. This modernizes legacy BPM by embedding resilient orchestration directly in application code.
