# Apigee

**Definition.** Apigee is an API management platform from Google Cloud that provides API gateway, security, analytics, and developer portal capabilities.
**Also known as:** Apigee Edge.

## Relationships
- API Management depends on Apigee.
- Apigee supersedes CA API Gateway.
- Apigee is related to Kong.

## Details
Apigee offers full-lifecycle API management including design, security policies, traffic management, monetization, and analytics, delivered as a managed cloud service on Google Cloud. It supports hybrid deployment for APIs running in different environments. This modernizes legacy gateways with cloud-scale management, rich analytics, and a developer-portal experience.
