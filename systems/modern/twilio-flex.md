# Twilio Flex

**Definition.** Twilio Flex is a programmable, API-first cloud contact-center platform from Twilio that layers omnichannel routing and a fully customisable agent desktop over existing systems.
**Also known as:** Flex, Twilio Flex CCaaS.

## Relationships
- Contact Center Platform depends on Twilio Flex.
- Twilio Flex supersedes Avaya Aura.

## Details
Twilio Flex is a programmable CCaaS platform built on Twilio's communications APIs, giving banks full control over the agent desktop, omnichannel routing logic, and integrations through code rather than fixed configuration. As a cloud-native, developer-extensible platform, it replaces rigid on-premise contact-center infrastructure with a composable voice and digital engagement layer.
