# Oracle Fusion ERP

**Definition.** Oracle Fusion ERP is a cloud-based enterprise resource planning suite from Oracle that provides general ledger, financial management, and reporting capabilities.
**Also known as:** Fusion Cloud ERP.

## Relationships
- General Ledger Engine depends on Oracle Fusion ERP.
- Oracle Fusion ERP supersedes SAP ECC.
- Oracle Fusion ERP is related to Workday Financials.

## Details
Oracle Fusion ERP delivers financials, procurement, and reporting as cloud SaaS with embedded analytics and AI-driven automation. Its unified data model supports continuous close and self-service reporting. This modernizes legacy on-premises ERP by providing regularly updated, cloud-native financial management with reduced infrastructure overhead.
