# Finxact

**Definition.** Finxact is a cloud-native core banking platform, now part of Fiserv, that delivers real-time deposit and lending processing through an API-first design.
**Also known as:** Finxact Core.

## Relationships
- Core Banking Processing depends on Finxact.
- Finxact supersedes FIS Profile.
- Finxact supersedes Infosys Finacle.
- Finxact is related to Mambu.

## Details
Finxact provides an open, programmable core built around a real-time ledger and a comprehensive API layer, deployed on public cloud. Its developer-oriented model lets banks and fintechs configure products and embed banking services. This modernizes legacy cores by exposing real-time data and extensibility that batch-based systems of record lack.
