# MuleSoft

**Definition.** MuleSoft is an integration and API platform from Salesforce, delivered as Anypoint Platform, that connects applications, data, and devices through APIs and integration flows.
**Also known as:** Anypoint Platform.

## Relationships
- Integration Platform depends on MuleSoft.
- MuleSoft supersedes Oracle SOA Suite.
- MuleSoft supersedes TIBCO BusinessWorks.
- MuleSoft is related to Boomi.

## Details
MuleSoft Anypoint Platform provides API-led connectivity with tooling to design, build, and manage APIs and integrations across hybrid and cloud environments. It emphasizes reusable APIs and an exchange of connectors. This modernizes monolithic enterprise service buses by promoting decoupled, reusable, API-based integration delivered as a managed platform.
