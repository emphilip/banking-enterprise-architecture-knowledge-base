# SaaScada

**Definition.** SaaScada is a cloud-native, data-driven core banking platform that uses CQRS and event-sourcing with a Product Sequencer to build multi-ledger, multi-currency current and savings deposit products without code.
**Also known as:** SaaScada Core, SaaScada Cloud Core.

## Relationships
- Core Banking Processing depends on SaaScada.
- SaaScada supersedes Jack Henry Core Director.

## Details
SaaScada is a cloud-native core built on CQRS and event-sourcing, exposing a Product Sequencer that lets institutions assemble multi-ledger, multi-currency current and savings products as configuration. Reviewed in Swisscom's Core Banking Radar, its data-driven, real-time model targets banks replacing legacy community and regional deposit cores with a flexible cloud platform.
