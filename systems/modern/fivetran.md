# Fivetran

**Definition.** Fivetran is a managed SaaS ELT and data-movement platform from Fivetran with pre-built connectors and automated schema management that loads source data into cloud warehouses.
**Also known as:** Fivetran ELT, Fivetran connectors.

## Relationships
- Data Warehousing depends on Fivetran.
- Fivetran supersedes Informatica PowerCenter.

## Details
Fivetran provides fully managed connectors that extract and load data from operational sources into cloud warehouses such as Snowflake, BigQuery, and Databricks, automatically handling schema drift and incremental syncs. By owning the extract-and-load tier, Fivetran removes hand-built ETL pipelines and pairs with in-warehouse transformation tools to complete the ELT pattern.
