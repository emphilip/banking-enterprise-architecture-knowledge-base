# Boomi

**Definition.** Boomi is a cloud-native integration platform as a service from Boomi that connects applications and data through low-code integration flows and API management.
**Also known as:** Dell Boomi.

## Relationships
- Integration Platform depends on Boomi.
- Boomi supersedes webMethods.
- Boomi is related to MuleSoft.

## Details
Boomi provides a multi-tenant iPaaS with a visual, low-code designer for building integrations, plus API management and data management capabilities. Delivered as a cloud service, it reduces the infrastructure burden of traditional middleware. This modernizes legacy on-premises integration suites by enabling faster, cloud-based connectivity with minimal coding.
