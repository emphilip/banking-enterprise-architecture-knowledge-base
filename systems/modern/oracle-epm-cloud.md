# Oracle EPM Cloud

**Definition.** Oracle EPM Cloud is a cloud enterprise performance management suite from Oracle (Financial Consolidation and Close, Planning, Narrative Reporting) that succeeds the on-premise Hyperion product line.
**Also known as:** Oracle EPM, Oracle Fusion EPM, FCCS, EPM Cloud.

## Relationships
- Financial Consolidation Platform depends on Oracle EPM Cloud.
- Oracle EPM Cloud supersedes Oracle Hyperion HFM.

## Details
Oracle EPM Cloud delivers Financial Consolidation and Close (FCCS), Planning, Account Reconciliation and Narrative Reporting as a cloud suite, automating group consolidation, currency translation, intercompany elimination and tagged statutory disclosure (XBRL). As the cloud successor to on-premise Hyperion it removes infrastructure and upgrade overhead while preserving the consolidation and close capability.
