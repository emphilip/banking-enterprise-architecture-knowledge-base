# Thought Machine Vault

**Definition.** Thought Machine Vault is a cloud-native core banking platform from Thought Machine that runs deposit, lending, and account products defined through configurable smart contracts.
**Also known as:** Vault, Vault Core.

## Relationships
- Core Banking Processing depends on Thought Machine Vault.
- Thought Machine Vault supersedes Temenos T24.
- Thought Machine Vault supersedes Oracle FLEXCUBE.
- Thought Machine Vault supersedes Avaloq Banking Suite.
- Thought Machine Vault is related to Mambu.

## Details
Thought Machine Vault is built on a microservices, API-first architecture and is typically deployed on public cloud or in containers using Kubernetes. Its smart-contract model lets banks define product logic in code, enabling rapid product launches and real-time processing. This cloud-native design modernizes monolithic legacy cores by replacing batch-oriented mainframe processing with elastic, continuously available services.
