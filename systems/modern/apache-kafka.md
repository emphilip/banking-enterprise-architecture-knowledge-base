# Apache Kafka

**Definition.** Apache Kafka is an open-source distributed event streaming platform from the Apache Software Foundation for publish/subscribe messaging, durable storage, and real-time processing of event streams.
**Also known as:** Apache Kafka Streaming.

## Relationships
- Data Streaming depends on Apache Kafka.
- Apache Kafka is related to Apache Flink.

## Details
Apache Kafka stores ordered, durable event logs partitioned across a cluster, decoupling producers from consumers and enabling replayable, high-throughput streaming. It serves as the de facto open-source backbone for event-driven architectures and real-time data pipelines, and is commercialized as a managed offering by Confluent. Banks use it for transaction event streams, fraud signals, and change-data-capture feeds.
