# SnapLogic

**Definition.** SnapLogic is a low-code cloud iPaaS from SnapLogic for application and data integration using pre-built connectors called Snaps, supporting ETL and ELT pipelines and API-led integration across SaaS and on-premises systems.
**Also known as:** SnapLogic iPaaS, SnapLogic Intelligent Integration Platform.

## Relationships
- Integration Platform depends on SnapLogic.
- SnapLogic supersedes IBM App Connect Enterprise.
- SnapLogic is related to Workato.

## Details
SnapLogic uses a library of pre-built connectors ("Snaps") and a visual pipeline designer to integrate SaaS and on-premises systems for both application and data integration. It supports ETL/ELT alongside API-led patterns. This modernizes on-premises ESB middleware by moving integration to a managed, low-code cloud platform.
