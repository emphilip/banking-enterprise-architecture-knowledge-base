# DataRobot

**Definition.** DataRobot is an enterprise AI platform from DataRobot that automates building, deploying, and monitoring machine learning models.
**Also known as:** DataRobot.

## Relationships
- Machine Learning Platform depends on DataRobot.
- DataRobot supersedes SAS Platform.
- DataRobot is related to Amazon SageMaker.

## Details
DataRobot provides automated machine learning (AutoML) along with MLOps capabilities to train, compare, deploy, and govern models, delivered as a cloud or self-managed platform. It accelerates model development for teams with varied data science expertise. This modernizes legacy analytics platforms by automating much of the model lifecycle and adding governance.
