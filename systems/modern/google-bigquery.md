# Google BigQuery

**Definition.** Google BigQuery is a serverless, cloud-native data warehouse from Google Cloud that runs large-scale SQL analytics without managing infrastructure.
**Also known as:** BigQuery.

## Relationships
- Data Warehousing depends on Google BigQuery.
- Google BigQuery supersedes IBM Db2.
- Google BigQuery is related to Snowflake.

## Details
Google BigQuery is a fully managed, serverless analytics platform that automatically scales compute and storage and supports standard SQL over very large datasets. It integrates built-in machine learning and streaming ingestion. This modernizes legacy on-premises databases and warehouses by eliminating provisioning and enabling fast, concurrent analytics on a consumption model.
