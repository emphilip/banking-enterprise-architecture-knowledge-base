# Hyperscience

**Definition.** Hyperscience is an intelligent document processing platform from Hyperscience that uses machine learning to classify documents and extract data from forms and unstructured content.
**Also known as:** Hyperscience.

## Relationships
- Intelligent Document Processing depends on Hyperscience.
- Hyperscience supersedes IBM FileNet.
- Hyperscience is related to AWS Textract.

## Details
Hyperscience applies machine learning to automate document classification and data extraction, combining automation with human-in-the-loop review to improve accuracy over time. It is delivered as software for cloud or on-premises deployment. This modernizes legacy content systems and manual data entry by automating ingestion of structured and unstructured documents.
