# Kong

**Definition.** Kong is a cloud-native API gateway and API management platform from Kong Inc that routes, secures, and observes API traffic across services.
**Also known as:** Kong Gateway.

## Relationships
- API Management depends on Kong.
- Kong supersedes CA API Gateway.
- Kong is related to Apigee.

## Details
Kong provides a lightweight, high-performance API gateway with a plugin architecture for authentication, rate limiting, and observability, well suited to microservices and Kubernetes. It can run self-hosted or as a managed service. This modernizes legacy API gateways by offering a container-friendly, extensible, cloud-native control plane for APIs.
