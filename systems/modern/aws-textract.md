# AWS Textract

**Definition.** AWS Textract is a machine learning service from Amazon Web Services that automatically extracts text, forms, and tables from scanned documents.
**Also known as:** Textract.

## Relationships
- Intelligent Document Processing depends on AWS Textract.
- AWS Textract supersedes OpenText Documentum.
- AWS Textract is related to Hyperscience.

## Details
AWS Textract uses pretrained machine learning models to perform OCR plus structured extraction of key-value pairs and tables, accessed through cloud APIs with no model management required. It scales on demand and integrates with other AWS services for downstream processing. This modernizes legacy content capture by replacing manual data entry with automated, API-driven document understanding.
