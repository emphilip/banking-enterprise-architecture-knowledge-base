# 10x Banking

**Definition.** 10x Banking is a cloud-native core banking platform from 10x Banking, marketed as SuperCore, that provides real-time deposit and lending processing for large banks.
**Also known as:** 10x, SuperCore.

## Relationships
- Core Banking Processing depends on 10x Banking.
- 10x Banking supersedes Legacy Mainframe Core.
- 10x Banking supersedes TCS BaNCS.
- 10x Banking is related to Thought Machine Vault.

## Details
10x Banking offers a metadata-driven, cloud-native core designed for tier-one banks, with real-time data, configurable products, and open APIs. It is delivered as a managed platform on public cloud infrastructure. By replacing batch mainframe cores with an event-driven, continuously available architecture, it enables real-time balances and faster product change.
