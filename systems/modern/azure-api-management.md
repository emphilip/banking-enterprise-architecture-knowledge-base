# Azure API Management

**Definition.** Azure API Management is a fully managed hybrid and multi-cloud API gateway and management service from Microsoft for publishing, securing, transforming, and analyzing APIs through a developer portal and policy engine.
**Also known as:** Azure APIM, APIM.

## Relationships
- API Management depends on Azure API Management.
- Azure API Management supersedes CA API Gateway.
- Azure API Management is related to AWS API Gateway.

## Details
Azure API Management provides a gateway, a publisher and admin plane, and a developer portal, with a policy engine for transformation, authentication, and rate limiting across hybrid and multi-cloud back ends. It supports self-hosted gateways for on-premises traffic. This modernizes legacy API gateways by combining managed cloud operation with hybrid reach.
