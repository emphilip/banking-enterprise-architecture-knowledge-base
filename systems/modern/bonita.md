# Bonita

**Definition.** Bonita is an open-source low-code BPM platform from Bonitasoft for designing, deploying, and running business processes and workflow applications, extensible via open-source connectors and deployable to Bonita Cloud.
**Also known as:** Bonitasoft, Bonita BPM.

## Relationships
- Workflow Orchestration depends on Bonita.
- Bonita supersedes IBM Business Automation Workflow.
- Bonita is related to Appian.

## Details
Bonita provides an open-source BPM engine, a process and UI designer, and a connector framework for building workflow-driven applications. It can run self-hosted or on Bonita Cloud. This modernizes proprietary on-premises BPM suites by offering an extensible open-source platform with cloud deployment.
