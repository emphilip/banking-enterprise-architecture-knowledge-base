# Gravitee

**Definition.** Gravitee is an open-source, event-native API management platform from Gravitee.io covering design, deployment, management, and security for both synchronous and asynchronous APIs through a gateway, console, and developer portal.
**Also known as:** Gravitee.io, Gravitee APIM.

## Relationships
- API Management depends on Gravitee.
- Gravitee supersedes CA API Gateway.
- Gravitee supersedes IBM API Connect.
- Gravitee is related to Kong.

## Details
Gravitee offers an Apache 2.0-licensed gateway, management console, and developer portal, with distinctive support for event-native and asynchronous APIs alongside REST. It can run self-hosted or with optional enterprise support. This modernizes heavyweight on-premises API management suites by adding event-driven API mediation and a cloud-native control plane.
