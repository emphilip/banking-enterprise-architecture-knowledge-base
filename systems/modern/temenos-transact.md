# Temenos Transact

**Definition.** Temenos Transact is a cloud-enabled core banking platform from Temenos that provides deposit, lending, and account processing as the successor to the T24 core.
**Also known as:** Transact.

## Relationships
- Core Banking Processing depends on Temenos Transact.
- Temenos Transact supersedes Temenos T24.
- Temenos Transact is related to Thought Machine Vault.

## Details
Temenos Transact evolves the long-established T24 core into a componentized, cloud-deployable platform with extensive packaged banking functionality and open APIs. It can run on public cloud or on-premises and supports SaaS delivery. This modernizes the legacy core by adding cloud elasticity, continuous deployment, and API-based extensibility on top of established banking capabilities.
