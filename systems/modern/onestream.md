# OneStream

**Definition.** OneStream is a unified cloud corporate performance management platform from OneStream Software combining financial close, consolidation, planning, reporting and disclosure on a single Extensible-Dimensionality model with built-in financial data quality.
**Also known as:** OneStream Software, OneStream Platform, OneStream XF.

## Relationships
- Financial Consolidation Platform depends on OneStream.
- OneStream supersedes Oracle Hyperion HFM.
- OneStream supersedes SAP BPC.

## Details
OneStream unifies the financial close, multi-entity/multi-GAAP consolidation (intercompany elimination, currency translation, ownership calculation), planning, budgeting and statutory/regulatory disclosure on one Extensible-Dimensionality model, with an embedded financial data quality layer that validates and certifies source data with full drill-back. As a single cloud CPM platform it replaces fragmented legacy consolidation and planning point solutions.
