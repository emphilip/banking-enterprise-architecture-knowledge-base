# Confluent Platform

**Definition.** Confluent Platform is an event streaming platform from Confluent, built on Apache Kafka, that provides real-time data streaming, integration, and stream processing.
**Also known as:** Confluent, Kafka.

## Relationships
- Data Streaming depends on Confluent Platform.
- Confluent Platform supersedes IBM MQ.
- Confluent Platform is related to Databricks.

## Details
Confluent Platform extends Apache Kafka with enterprise tooling for connectors, schema management, and stream processing, available both self-managed and as a fully managed cloud service. Its distributed log architecture supports high-throughput, durable event streaming. This modernizes traditional message queues by enabling scalable, replayable, real-time event pipelines.
