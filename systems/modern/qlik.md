# Qlik

**Definition.** Qlik is a self-service business intelligence and data analytics platform from Qlik, built on a patented in-memory associative engine that enables non-query-based data exploration and governed dashboards through Qlik Sense.
**Also known as:** Qlik Sense, QlikView, Qlik Cloud.

## Relationships
- Analytics Platform depends on Qlik.
- Qlik supersedes SAP BusinessObjects.

## Details
Qlik's associative engine holds data in memory and lets users freely navigate relationships without predefining query paths, surfacing both related and unrelated values during exploration. Qlik Sense provides governed self-service dashboards and Qlik Cloud delivers SaaS deployment. The associative model differentiates Qlik from purely query-based BI tools.
