# Amazon SageMaker

**Definition.** Amazon SageMaker is a cloud-based machine learning platform from Amazon Web Services that provides tools to build, train, and deploy machine learning models at scale.
**Also known as:** SageMaker.

## Relationships
- Machine Learning Platform depends on Amazon SageMaker.
- Amazon SageMaker supersedes SAS Platform.
- Amazon SageMaker is related to Google Vertex AI.

## Details
Amazon SageMaker offers a fully managed environment covering the machine learning lifecycle, including data labeling, training, tuning, deployment, and monitoring, with elastic cloud compute. It supports MLOps practices and managed inference endpoints. This modernizes legacy analytics platforms by providing scalable, cloud-native model development and deployment.
