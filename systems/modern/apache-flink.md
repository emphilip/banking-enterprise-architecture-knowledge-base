# Apache Flink

**Definition.** Apache Flink is an open-source stateful stream-processing engine from the Apache Software Foundation for low-latency, high-throughput real-time analytics and complex event processing over data in motion.
**Also known as:** Flink, Managed Service for Apache Flink.

## Relationships
- Data Streaming depends on Apache Flink.
- Apache Flink is related to Apache Kafka.

## Details
Apache Flink processes unbounded and bounded data streams with exactly-once state consistency, event-time semantics, and windowing for continuous computation. It is commonly paired with streaming sources such as Kafka and Kinesis to power real-time aggregations, alerting, and complex event processing, and is available as a managed cloud service. Banks apply it to real-time fraud scoring and instant-payment analytics.
