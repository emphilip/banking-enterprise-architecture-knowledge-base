# AWS API Gateway

**Definition.** AWS API Gateway is a fully managed cloud service from Amazon Web Services to create, publish, maintain, monitor, and secure REST, HTTP, and WebSocket APIs at scale with throttling, authorization, and usage plans.
**Also known as:** Amazon API Gateway, APIGW.

## Relationships
- API Management depends on AWS API Gateway.
- AWS API Gateway supersedes CA API Gateway.
- AWS API Gateway is related to Azure API Management.

## Details
AWS API Gateway acts as a managed front door for serverless and microservice backends, handling request authorization, throttling, caching, and usage plans without operating gateway infrastructure. It integrates natively with AWS Lambda and other AWS services. This modernizes self-managed, on-premises API gateways by providing an elastic, fully managed control plane.
