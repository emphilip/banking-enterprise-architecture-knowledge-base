# Snowflake

**Definition.** Snowflake is a cloud-native data warehousing and analytics platform from Snowflake that separates storage and compute to deliver elastic SQL analytics across clouds.
**Also known as:** Snowflake Data Cloud.

## Relationships
- Data Warehousing depends on Snowflake.
- Snowflake supersedes Teradata.
- Snowflake is related to Databricks.

## Details
Snowflake provides a fully managed, multi-cloud data platform with a decoupled storage-and-compute architecture that scales virtual warehouses independently and on demand. It supports structured and semi-structured data, secure data sharing, and pay-per-use pricing. This modernizes legacy on-premises appliances by removing capacity planning and enabling elastic, concurrent analytics.
