# Google Vertex AI

**Definition.** Google Vertex AI is a cloud-based machine learning platform from Google Cloud that unifies tools to build, train, deploy, and manage machine learning and generative AI models.
**Also known as:** Vertex AI.

## Relationships
- Machine Learning Platform depends on Google Vertex AI.
- Google Vertex AI supersedes SAS Platform.
- Google Vertex AI is related to Amazon SageMaker.

## Details
Google Vertex AI provides a managed, end-to-end ML platform with training, pipelines, feature store, model registry, and access to foundation models for generative AI. It runs on Google Cloud with elastic compute and MLOps tooling. This modernizes legacy analytics platforms by offering integrated, cloud-native model development and deployment.
