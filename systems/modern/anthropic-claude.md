# Anthropic Claude

**Definition.** Anthropic Claude is a family of large language models from Anthropic that powers generative AI applications such as conversation, summarization, and content generation.
**Also known as:** Claude.

## Relationships
- Generative AI Platform depends on Anthropic Claude.
- Anthropic Claude supersedes Legacy IVR System.
- Anthropic Claude is related to Kasisto.

## Details
Anthropic Claude provides cloud-accessible large language models offered via API and through cloud marketplaces, supporting natural-language understanding, reasoning, tool use, and long-context tasks. Anthropic develops Claude with an emphasis on safety and reliability. This modernizes legacy automated interaction systems by enabling flexible, natural-language generative AI for customer servicing and advisory use cases.
