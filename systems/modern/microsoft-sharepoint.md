# Microsoft SharePoint

**Definition.** Microsoft SharePoint is an enterprise content and document management and collaboration platform from Microsoft, offering versioning, metadata, records retention, and workflow, deeply integrated with Microsoft 365.
**Also known as:** SharePoint Online, SharePoint Server, M365 SharePoint.

## Relationships
- Document Management depends on Microsoft SharePoint.
- Microsoft SharePoint supersedes IBM FileNet.

## Details
Microsoft SharePoint manages documents and content in libraries with versioning, metadata columns, retention labels, and approval workflows, available as SharePoint Online in Microsoft 365 or as on-premises SharePoint Server. Its tight integration with Microsoft 365 productivity tools makes it a common content services and collaboration layer in banks, including for document-centric onboarding and case files.
