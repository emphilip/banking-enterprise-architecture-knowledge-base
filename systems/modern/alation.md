# Alation

**Definition.** Alation is an enterprise data catalog and governance platform from Alation that uses an active metadata graph for data discovery, stewardship, policy enforcement, and access controls across hybrid and multi-cloud estates.
**Also known as:** Alation Data Catalog, Alation Data Governance.

## Relationships
- Data Governance depends on Alation.
- Alation supersedes Informatica Axon.

## Details
Alation builds an active metadata graph that captures how data is used, by whom, and in which queries, then surfaces that behavioral context to drive discovery and trustworthy reuse. Its governance capabilities layer policies, stewardship, and access controls over catalogued assets. Alation targets self-service analytics enablement alongside enterprise governance.
