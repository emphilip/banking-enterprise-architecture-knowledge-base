# Galileo

**Definition.** Galileo is a cloud-based card issuing and payments processing platform from Galileo Financial Technologies that provides APIs for issuing cards and managing accounts.
**Also known as:** Galileo FT.

## Relationships
- Card Processing depends on Galileo.
- Galileo supersedes First Data Cards.
- Galileo is related to Marqeta.

## Details
Galileo provides a programmable platform for card issuing, payments, and account management used by fintechs and banks, delivered through open APIs on cloud infrastructure. It supports virtual and physical cards, authorization controls, and ledgering. This modernizes legacy card platforms by offering real-time, API-driven program management instead of batch processing.
