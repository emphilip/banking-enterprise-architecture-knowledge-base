# Teradata

**Definition.** Teradata is an enterprise data warehouse platform from Teradata used by banks for large-scale analytics, reporting, and consolidated data storage.
**Also known as:** Teradata EDW.

## Relationships
- Data Warehousing depends on Teradata.
- Teradata mentions Snowflake.
- Teradata is related to Oracle Exadata.

## Details
Teradata is a massively parallel processing (MPP) data warehouse historically deployed as on-premises appliances for high-volume enterprise analytics. It has long served as the central enterprise data warehouse at large banks for risk, finance, and customer analytics. Cloud data platforms now compete strongly with traditional appliance deployments, driving migration projects.
