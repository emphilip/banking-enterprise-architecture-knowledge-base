# TIBCO BusinessWorks

**Definition.** TIBCO BusinessWorks is an integration and enterprise service bus platform from TIBCO used to connect applications and orchestrate data flows across systems.
**Also known as:** BusinessWorks, TIBCO EMS.

## Relationships
- Integration Platform depends on TIBCO BusinessWorks.
- TIBCO BusinessWorks is related to webMethods.

## Details
TIBCO BusinessWorks provides service orchestration and application integration, frequently paired with the TIBCO EMS messaging layer. It is traditionally deployed on-premises as part of an enterprise service bus architecture connecting core, channel, and back-office systems. Cloud-native integration and API platforms increasingly succeed classic ESB deployments.
