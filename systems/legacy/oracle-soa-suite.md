# Oracle SOA Suite

**Definition.** Oracle SOA Suite is a service-oriented architecture middleware platform from Oracle used to build, deploy, and orchestrate integration services and composite applications.
**Also known as:** SOA Suite.

## Relationships
- Integration Platform depends on Oracle SOA Suite.
- Oracle SOA Suite mentions MuleSoft.
- Oracle SOA Suite is related to webMethods.

## Details
Oracle SOA Suite provides a service bus, BPEL-based orchestration, and adapters for integrating heterogeneous systems, deployed on Oracle Fusion Middleware. Banks use it to expose and compose services across core and back-office applications. API-led and cloud integration platforms increasingly succeed traditional SOA Suite deployments.
