# CA API Gateway

**Definition.** CA API Gateway is an API gateway and management product, originally Layer7 and now part of Broadcom, used to secure, route, and govern API traffic.
**Also known as:** Layer7, CA Layer7.

## Relationships
- API Management depends on CA API Gateway.
- CA API Gateway mentions Kong.
- CA API Gateway is related to Oracle SOA Suite.

## Details
CA API Gateway (formerly Layer7) provides API security, throttling, transformation, and policy enforcement, typically deployed on-premises at the network edge. Banks use it to expose internal services to digital channels and partners. Lightweight, cloud-native API gateways increasingly compete with established gateway deployments.
