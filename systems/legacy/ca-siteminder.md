# CA SiteMinder

**Definition.** CA SiteMinder is a web access management and single sign-on product, now part of Broadcom, used to authenticate users and enforce access policies for web applications.
**Also known as:** SiteMinder.

## Relationships
- Identity Access Management depends on CA SiteMinder.
- CA SiteMinder mentions Okta.
- CA SiteMinder is related to Oracle Identity Manager.

## Details
CA SiteMinder provides web single sign-on, authentication, and authorization policy enforcement, traditionally deployed on-premises in front of enterprise web applications. Banks use it to centralize access control for employee and customer portals. Cloud identity platforms increasingly supersede on-premises web access management deployments.
