# Oracle Hyperion HFM

**Definition.** Oracle Hyperion HFM is a legacy on-premise Hyperion Financial Management product from Oracle for group financial consolidation and close, long the incumbent that is now being migrated to cloud EPM.
**Also known as:** HFM, Hyperion Financial Management, Hyperion.

## Relationships
- Financial Consolidation Platform depends on Oracle Hyperion HFM.
- Oracle Hyperion HFM is related to SAP BPC.

## Details
Oracle Hyperion HFM performs multi-entity, multi-GAAP group consolidation, intercompany elimination, currency translation, minority-interest calculation and statutory close on an on-premise Hyperion stack. As a long-standing financial-consolidation incumbent it is widely deployed for legal and management consolidation, and Oracle now positions Oracle EPM Cloud as its cloud successor.
