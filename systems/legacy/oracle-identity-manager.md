# Oracle Identity Manager

**Definition.** Oracle Identity Manager is an identity governance and user provisioning product from Oracle used to manage user accounts, roles, and access across enterprise systems.
**Also known as:** OIM.

## Relationships
- Identity Access Management depends on Oracle Identity Manager.
- Oracle Identity Manager mentions ForgeRock.
- Oracle Identity Manager is related to CA SiteMinder.

## Details
Oracle Identity Manager (OIM) provides user lifecycle management, provisioning, and access certification, traditionally deployed on-premises as part of Oracle Identity Management. Banks use it to automate joiner-mover-leaver processes and enforce access governance. Cloud-based identity governance platforms increasingly succeed on-premises OIM deployments.
