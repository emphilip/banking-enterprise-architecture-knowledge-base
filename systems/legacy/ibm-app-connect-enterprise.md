# IBM App Connect Enterprise

**Definition.** IBM App Connect Enterprise is an enterprise integration and ESB runtime from IBM, formerly IBM Integration Bus and WebSphere Message Broker, for routing and transforming messages between disparate applications.
**Also known as:** IBM ACE, IBM Integration Bus, IIB, WebSphere Message Broker.

## Relationships
- Integration Platform depends on IBM App Connect Enterprise.
- IBM App Connect Enterprise is related to IBM MQ.

## Details
IBM App Connect Enterprise (the successor to IBM Integration Bus and WebSphere Message Broker) provides message routing, transformation, and protocol mediation across service-oriented architectures. It is long-standing on-premises middleware in banks integrating core systems with channels and partners. Cloud iPaaS offerings increasingly take over new integration workloads.
