# LexisNexis Bridger Insight

**Definition.** LexisNexis Bridger Insight is a watchlist and sanctions screening platform from LexisNexis Risk Solutions (Fircosoft) that screens customers and payments against sanctions, PEP and adverse-media data using fuzzy name matching.
**Also known as:** Bridger Insight XG, Fircosoft, Firco, Firco Continuity.

## Relationships
- Transaction Monitoring Platform depends on LexisNexis Bridger Insight.
- LexisNexis Bridger Insight is related to Oracle Mantas.

## Details
LexisNexis Bridger Insight (Bridger Insight XG, with Firco Compliance Link / Continuity) performs name and entity screening against sanctions, PEP, adverse-media and internal watchlists, using configurable fuzzy-matching algorithms and hit-disposition workflows to manage false positives. It draws on LexisNexis WorldCompliance reference data and is typically deployed as an established, on-premises screening engine that modern cloud-native financial-crime platforms increasingly supersede.
