# Oracle E-Business Suite

**Definition.** Oracle E-Business Suite is an on-premise ERP financials suite from Oracle (Oracle GL, AP, AR and Fixed Assets) widely deployed as a legacy corporate general ledger before the move to cloud ERP.
**Also known as:** Oracle EBS, Oracle Financials, E-Business Suite.

## Relationships
- General Ledger Engine depends on Oracle E-Business Suite.
- Oracle E-Business Suite is related to SAP ECC.

## Details
Oracle E-Business Suite delivers integrated General Ledger, subledger accounting (Accounts Payable, Accounts Receivable, Fixed Assets) and financial reporting on a single on-premise application stack. It served as the system of record for the corporate chart of accounts, journal posting and period-end close for many large enterprises prior to cloud ERP adoption, and is now commonly migrated to Oracle Fusion ERP or SAP S/4HANA Finance.
