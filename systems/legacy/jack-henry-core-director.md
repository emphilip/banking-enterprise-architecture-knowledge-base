# Jack Henry Core Director

**Definition.** Jack Henry Core Director is Jack Henry's in-house community-bank deposit and loan core platform, positioned as an alternative to SilverLake for smaller institutions.
**Also known as:** Core Director, Jack Henry Core Director System.

## Relationships
- Core Banking Processing depends on Jack Henry Core Director.
- Jack Henry Core Director is related to Jack Henry SilverLake.

## Details
Jack Henry Core Director serves smaller community banks with deposit and loan account servicing, posting and reporting, and is one of Jack Henry's distinct core offerings noted in the Kansas City Fed market-structure brief and bank-tech provider listings. It sits alongside the larger SilverLake platform in the vendor's core portfolio and is a candidate for migration to cloud-native deposit cores.
