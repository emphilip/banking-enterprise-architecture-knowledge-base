# SAP BusinessObjects

**Definition.** SAP BusinessObjects is an enterprise business intelligence and reporting suite from SAP, providing pixel-perfect reporting, ad-hoc analysis, and dashboards through tools such as Web Intelligence and Crystal Reports over semantic universes.
**Also known as:** BOBJ, BusinessObjects BI, Web Intelligence, Crystal Reports.

## Relationships
- Analytics Platform depends on SAP BusinessObjects.
- SAP BusinessObjects is related to IBM Cognos.

## Details
SAP BusinessObjects is a long-standing on-premises BI incumbent built around the semantic-layer "universe" abstraction that shields report authors from underlying schemas. It bundles Web Intelligence for ad-hoc query and analysis and Crystal Reports for formatted, pixel-perfect output. SAP positions BusinessObjects as a mature platform now approaching end of mainstream maintenance, with customers migrating toward cloud BI.
