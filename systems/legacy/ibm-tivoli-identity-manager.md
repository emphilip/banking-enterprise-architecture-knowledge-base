# IBM Tivoli Identity Manager

**Definition.** IBM Tivoli Identity Manager is an identity management and provisioning product from IBM, later known as IBM Security Identity Manager, used to administer user identities and access rights.
**Also known as:** Tivoli Identity Manager, ISIM.

## Relationships
- Customer Identity depends on IBM Tivoli Identity Manager.
- IBM Tivoli Identity Manager mentions Ping Identity.
- IBM Tivoli Identity Manager is related to Oracle Identity Manager.

## Details
IBM Tivoli Identity Manager (later IBM Security Identity Manager, ISIM) provides automated user provisioning, role management, and access governance, traditionally deployed on-premises. Banks have used it to manage identities across heterogeneous systems. Modern cloud identity and CIAM platforms increasingly replace these legacy identity suites.
