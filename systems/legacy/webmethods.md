# webMethods

**Definition.** webMethods is an integration platform from Software AG used to connect applications, orchestrate business processes, and manage APIs across the enterprise.
**Also known as:** Software AG webMethods.

## Relationships
- Integration Platform depends on webMethods.
- webMethods mentions Boomi.
- webMethods is related to TIBCO BusinessWorks.

## Details
webMethods provides enterprise application integration, an integration server, and API management, traditionally deployed on-premises as an ESB. Banks use it to integrate core banking, channels, and partner systems. Modern iPaaS and API-led platforms increasingly replace classic webMethods estates during integration modernization.
