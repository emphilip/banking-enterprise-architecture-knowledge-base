# FIS Profile

**Definition.** FIS Profile is a real-time core banking system from FIS that manages deposit accounts, loans, and customer records for retail and commercial banks.
**Also known as:** Profile.

## Relationships
- Core Banking Processing depends on FIS Profile.
- FIS Profile mentions Finxact.
- FIS Profile is related to Fiserv DNA.

## Details
FIS Profile is typically deployed on-premises or in private data centers and has long served as a system of record for account and ledger processing at large institutions. As a long-established platform, it is frequently a modernization target, with banks migrating toward cloud-native cores. Its real-time architecture distinguishes it from older batch-oriented mainframe cores.
