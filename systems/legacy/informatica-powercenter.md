# Informatica PowerCenter

**Definition.** Informatica PowerCenter is a classic on-premises ETL engine from Informatica that extracts source data, transforms it in a dedicated mid-tier service, and loads it into enterprise data warehouses.
**Also known as:** PowerCenter, Informatica ETL.

## Relationships
- Data Warehousing depends on Informatica PowerCenter.
- Informatica PowerCenter is related to Teradata.

## Details
Informatica PowerCenter is the incumbent integration tier in many bank data estates, executing mapping-based transformations on a dedicated server before loading into the warehouse. This ETL pattern is now being decomposed into modern ELT, where extraction and loading move data quickly into the warehouse and transformation runs in-warehouse. Informatica directs new development toward its cloud data integration services.
