# IBM API Connect

**Definition.** IBM API Connect is an on-premises and hybrid full-lifecycle API management suite from IBM for creating, securing, managing, socializing, and analyzing APIs through a gateway, API manager, and developer portal.
**Also known as:** API Connect, IBM APIC.

## Relationships
- API Management depends on IBM API Connect.
- IBM API Connect is related to CA API Gateway.

## Details
IBM API Connect packages an API gateway (DataPower-based), an API manager, and a developer portal into a single full-lifecycle product. It has long been the API management incumbent in regulated enterprises and banks that run on-premises or hybrid deployments. Cloud-native and SaaS API platforms increasingly displace heavyweight gateway suites.
