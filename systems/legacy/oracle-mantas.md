# Oracle Mantas

**Definition.** Oracle Mantas is an anti-money-laundering and transaction monitoring platform from Oracle, now part of Oracle Financial Crime and Compliance Management, used to detect suspicious financial activity.
**Also known as:** Mantas, Oracle FCCM.

## Relationships
- Transaction Monitoring Platform depends on Oracle Mantas.
- Oracle Mantas mentions Hawk AI.
- Oracle Mantas is related to NICE Actimize.

## Details
Oracle Mantas, the basis of Oracle Financial Crime and Compliance Management (FCCM), provides behavior detection, transaction monitoring, and alert management for AML programs. It is traditionally deployed on-premises against enterprise data stores. Cloud-native, AI-driven monitoring platforms increasingly compete with established AML systems.
