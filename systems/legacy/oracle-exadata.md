# Oracle Exadata

**Definition.** Oracle Exadata is an engineered database system from Oracle that runs the Oracle Database on optimized hardware for high-performance transaction processing and analytics.
**Also known as:** Exadata.

## Relationships
- Data Warehousing depends on Oracle Exadata.
- Oracle Exadata mentions Databricks.
- Oracle Exadata is related to Teradata.

## Details
Oracle Exadata is a tightly integrated hardware-and-software platform tuned to run Oracle Database workloads, deployed on-premises and via Oracle Cloud. Banks use it to host high-throughput databases and data warehouses underpinning core and analytics systems. As cloud lakehouse and warehouse platforms mature, Exadata workloads are common modernization candidates.
