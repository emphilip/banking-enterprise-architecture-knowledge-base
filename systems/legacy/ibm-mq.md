# IBM MQ

**Definition.** IBM MQ is a message-oriented middleware product from IBM that provides reliable, asynchronous messaging between applications using queues.
**Also known as:** WebSphere MQ, MQSeries.

## Relationships
- Data Streaming depends on IBM MQ.
- IBM MQ mentions Confluent Platform.
- IBM MQ is related to TIBCO BusinessWorks.

## Details
IBM MQ, formerly WebSphere MQ and originally MQSeries, provides assured, transactional message delivery widely used to integrate banking applications across mainframe and distributed systems. It is deployed on-premises across many platforms and is deeply embedded in payment and core integration flows. Event-streaming platforms now address many use cases previously served by queue-based messaging.
