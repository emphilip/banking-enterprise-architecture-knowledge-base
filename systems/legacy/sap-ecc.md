# SAP ECC

**Definition.** SAP ECC is the SAP ERP Central Component, an enterprise resource planning system from SAP that includes financial accounting and general ledger functionality.
**Also known as:** SAP ERP, SAP R/3.

## Relationships
- General Ledger Engine depends on SAP ECC.
- SAP ECC mentions Workday Financials.
- SAP ECC is related to Oracle FLEXCUBE.

## Details
SAP ECC (the successor to SAP R/3) is a widely deployed ERP providing general ledger, financial accounting, and back-office functions, typically run on-premises. Banks use it for corporate finance, accounting, and reporting. SAP has positioned S/4HANA as the successor, and cloud ERP platforms increasingly compete during finance modernization.
