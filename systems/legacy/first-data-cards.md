# First Data Cards

**Definition.** First Data Cards refers to the card issuing and processing platforms from First Data, now part of Fiserv, including the VisionPLUS card management system used by issuers worldwide.
**Also known as:** First Data, VisionPLUS.

## Relationships
- Card Processing depends on First Data Cards.
- First Data Cards mentions Galileo.
- First Data Cards is related to TSYS TS2.

## Details
First Data Cards (notably the VisionPLUS platform) provides card account management, authorization, and statement processing for credit, debit, and private-label programs. The platform has a long mainframe-based heritage and is widely licensed and operated by banks and processors internationally. Following First Data's acquisition by Fiserv, it remains a major incumbent card system alongside newer cloud-native issuer-processor platforms.
