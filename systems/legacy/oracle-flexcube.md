# Oracle FLEXCUBE

**Definition.** Oracle FLEXCUBE is a core banking application from Oracle that supports retail, corporate, and Islamic banking account and transaction processing for banks globally.
**Also known as:** FLEXCUBE.

## Relationships
- Core Banking Processing depends on Oracle FLEXCUBE.
- Oracle FLEXCUBE is related to Temenos T24.

## Details
Oracle FLEXCUBE is a widely adopted core banking platform, particularly in emerging markets and across the Middle East and Asia. It is typically deployed on-premises on Oracle database infrastructure and covers deposits, lending, and payments functions. Oracle has positioned newer cloud banking offerings alongside FLEXCUBE as institutions modernize.
