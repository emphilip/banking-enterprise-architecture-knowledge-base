# Integration & APIs

**Definition.** Integration & APIs is the technology domain covering API management, integration platforms, workflow orchestration, and notification services that connect systems and expose banking services.
**Also known as:** Integration Domain, API Domain.

## Relationships
- API Management is defined in the Integration & APIs domain.
- Integration Platform is defined in the Integration & APIs domain.
- Workflow Orchestration is defined in the Integration & APIs domain.
- Notification Services is defined in the Integration & APIs domain.
- Integration & APIs is related to the Core Processing domain.

## Details
The Integration & APIs domain provides API gateways, enterprise integration (ESB/iPaaS), business process and workflow orchestration, and messaging/notification services. It underpins Open Banking and embedded finance, with modernization moving from legacy ESBs to cloud-native API gateways such as Kong and Apigee.
