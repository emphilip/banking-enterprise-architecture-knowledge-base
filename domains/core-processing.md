# Core Processing

**Definition.** Core Processing is the technology domain covering the foundational transaction-processing engines of the bank, including the core banking ledger, payment, card, lending, and general ledger systems.
**Also known as:** Core Systems Domain.

## Relationships
- Core Banking Processing is defined in the Core Processing domain.
- Payment Orchestration is defined in the Core Processing domain.
- Card Processing is defined in the Core Processing domain.
- Loan Origination Platform is defined in the Core Processing domain.
- General Ledger Engine is defined in the Core Processing domain.
- Core Processing is related to the Integration & APIs domain.

## Details
The Core Processing domain provides the systems of record that realize business capabilities across deposits, payments, cards, and lending, including core banking processing, payment orchestration, card processing, and credit decisioning engines. Modernization here shifts from legacy mainframe cores to cloud-native platforms such as Thought Machine Vault and Mambu.
