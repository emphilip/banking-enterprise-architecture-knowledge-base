# Threat Detection

**Definition.** Threat Detection is the technology capability that collects security telemetry, correlates events, and identifies cyber threats and anomalies to support security operations and incident response.
**Also known as:** SIEM, Cybersecurity.

## Relationships
- Threat Detection is defined in the Security & Identity domain.
- Threat Detection depends on the Data Streaming capability.
- Threat Detection depends on the Identity Access Management capability.
- Threat Detection is related to Fraud Analytics.

## Details
Threat Detection aggregates logs, identity events, and network telemetry into a SIEM, applying rules and machine learning to surface intrusions and operational risk events. It consumes high-volume event feeds via the Data Streaming capability and identity signals from Identity Access Management. Modern security operations use cloud-native SIEM and XDR with automated response playbooks. It supports operational risk management within the Security & Identity domain.
