# Loss Event Register

**Definition.** Loss Event Register is the technology capability that captures, classifies and tracks operational-risk loss events and near-misses for analysis, scenario calibration and operational-risk capital.
**Also known as:** Loss Database, Operational Loss Register, Loss Event Capture.

## Relationships
- Loss Event Register is defined in the Data & Analytics domain.
- Loss Event Register is derived from Governance Risk & Compliance Platform.
- Loss Event Register is related to Risk Control Self-Assessment Engine.
- Loss Event Register is related to Economic Capital Engine.

## Details
Loss Event Register records internal loss events and near-misses with gross and net loss amounts, recoveries, Basel event-type and business-line classification, and root-cause and remediation tracking. The accumulated internal loss data, combined with external loss data, calibrates operational-risk scenario analysis and feeds operational-risk capital models, as delivered by IBM OpenPages and MetricStream.
