# Skills-Based Routing

**Definition.** Skills-Based Routing is the technology sub-capability providing routing logic that matches interactions to agents by skill, proficiency and priority across the universal queue.
**Also known as:** Skill Routing, Proficiency Routing.

## Relationships
- Skills-Based Routing is defined in the Channels & Engagement domain.
- Skills-Based Routing is derived from Omnichannel Routing Engine.

## Details
Skills-Based Routing tags each interaction with required skills (product, language, segment) and each agent with rated proficiencies, then selects the most suitable available agent, relaxing constraints over time so service levels hold under load. It supports priority weighting, bullseye/ring expansion of eligible agent pools and last-agent routing for continuity. Amazon Connect and Genesys implement skills and proficiency as the matching rules layered on top of the omnichannel routing engine.
