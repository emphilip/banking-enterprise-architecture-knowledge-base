# Model Portfolio Manager

**Definition.** Model Portfolio Manager is the technology capability that authors, versions and distributes model portfolios and target asset allocations, links accounts to models and sleeves, and feeds target weights to rebalancing and trade-order generation.
**Also known as:** Model Management Service, Target Allocation Manager, Sleeve & Model Engine.

## Relationships
- Model Portfolio Manager is defined in the Core Processing domain.
- Model Portfolio Manager is derived from Portfolio Management System.
- Model Portfolio Manager is related to Portfolio Management.
- Model Portfolio Manager is related to Rebalancing Engine.

## Details
Model Portfolio Manager is where strategists define and version target asset allocations and model portfolios, then map accounts and sleeves to those models in unified managed account (UMA) and SMA structures. It publishes target weights that the Rebalancing Engine consumes to compute drift and generate orders. Platforms such as Envestnet, Charles River IMS and Bloomberg buy-side realize model authoring, sleeve management and model-to-account linkage, supporting overlay management and tactical tilts across large advisor books.
