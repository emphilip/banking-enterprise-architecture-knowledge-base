# Data Streaming

**Definition.** Data Streaming is the technology capability that captures, transports, and processes continuous event streams in real time, enabling event-driven architectures and low-latency analytics.
**Also known as:** Event Streaming.

## Relationships
- Data Streaming is defined in the Data & Analytics domain.
- Data Streaming mentions IBM MQ.
- Data Streaming mentions Confluent Platform.
- Data Streaming is related to Data Warehousing.
- Data Streaming is related to Fraud Analytics.

## Details
Data Streaming underpins real-time payments, fraud scoring, and event sourcing by moving data as it is produced rather than in batch. Legacy messaging such as IBM MQ provides point-to-point queues, while the modern Confluent Platform delivers a distributed, cloud-native Kafka log with stream processing and connectors. It is the event backbone linking core, channels, and analytics in a modern reference architecture.
