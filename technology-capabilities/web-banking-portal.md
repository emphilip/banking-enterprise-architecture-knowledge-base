# Web Banking Portal

**Definition.** Web Banking Portal is the technology sub-capability delivering the browser-based responsive and progressive-web-app banking web application and widget framework for retail and business users.
**Also known as:** Internet Banking Portal, Web Channel, Online Banking Portal.

## Relationships
- Web Banking Portal is defined in the Channels & Engagement domain.
- Web Banking Portal is derived from Digital Channel Platform.
- Web Banking Portal depends on API Management.

## Details
Web Banking Portal renders responsive layouts and component-based widgets across desktop and mobile browsers, supporting both retail self-service and entitlement-rich business banking (multi-user, approvals, role-based access). It is API-driven, consuming the same managed services as the mobile channel for consistency, and supports white-labelling and progressive enhancement. Platforms such as Backbase Engagement Banking and Alkami expose configurable widget catalogues so institutions assemble portals without bespoke front-end builds.
