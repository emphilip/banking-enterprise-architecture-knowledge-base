# SWIFT Connectivity Adapter

**Definition.** SWIFT Connectivity Adapter is the technology capability providing SWIFT-network connectivity (including gpi/Go, FIN and InterAct/FileAct) feeding the scheme gateway.
**Also known as:** SWIFT Adapter, SWIFT gpi Connector.

## Relationships
- SWIFT Connectivity Adapter is defined in the Core Processing domain.
- SWIFT Connectivity Adapter is derived from Scheme Connectivity Gateway.
- SWIFT Connectivity Adapter is related to Payment Message Transformer.

## Details
SWIFT Connectivity Adapter handles session management, security and message exchange over the SWIFT network across FIN, InterAct and FileAct, and supports gpi/Go tracking for cross-border payments. It supplies SWIFT-specific connectivity into the broader scheme gateway, as seen in ACI Enterprise Payments Platform.
