# Machine Learning Platform

**Definition.** Machine Learning Platform is the technology capability that supports the full model lifecycle—feature engineering, training, deployment, monitoring, and governance—for predictive and analytical models across the bank.
**Also known as:** MLOps, ML Platform.

## Relationships
- Machine Learning Platform is defined in the AI & Automation domain.
- Machine Learning Platform depends on the Data Warehousing capability.
- Machine Learning Platform depends on the Data Streaming capability.
- Machine Learning Platform mentions SAS Platform.
- Machine Learning Platform mentions Amazon SageMaker.
- Machine Learning Platform mentions Google Vertex AI.
- Machine Learning Platform mentions DataRobot.

## Details
Machine Learning Platform provides feature stores, training pipelines, model registries, and serving infrastructure with monitoring for drift and bias. Legacy analytics suites such as the SAS Platform bundle modelling but lack modern MLOps, while cloud platforms like Amazon SageMaker, Google Vertex AI, and DataRobot offer scalable, governed pipelines. It powers credit decisioning, fraud, and transaction monitoring use cases and depends on warehoused and streamed data.
