# Application Intake Engine

**Definition.** Application Intake Engine is the technology capability that captures and validates inbound loan applications across channels, manages application state, runs eligibility pre-checks and routes applications into the origination workflow.
**Also known as:** Application Capture Engine, Origination Intake Service.

## Relationships
- Application Intake Engine is defined in the Core Processing domain.
- Application Intake Engine is derived from Loan Origination Platform.
- Application Intake Engine is related to Application Processing.

## Details
Application Intake Engine provides omni-channel application capture, field-level validation, duplicate detection and routing into the loan origination workflow. Modern loan origination suites such as nCino and Temenos Infinity Origination expose configurable intake forms that maintain application state and pre-qualify applicants before underwriting, supporting straight-through origination across consumer, commercial and mortgage lending.
