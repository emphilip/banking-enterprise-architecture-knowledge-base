# Robotic Process Automation

**Definition.** Robotic Process Automation is the technology capability that automates repetitive, rule-based tasks by emulating human interactions with user interfaces and applications via software bots.
**Also known as:** RPA.

## Relationships
- Robotic Process Automation is defined in the AI & Automation domain.
- Robotic Process Automation depends on the Workflow Orchestration capability.
- Robotic Process Automation is related to Intelligent Document Processing.

## Details
Robotic Process Automation bridges legacy systems that lack APIs by scripting UI-level interactions for tasks such as reconciliation, data entry, and account servicing. It is typically a tactical accelerator that coordinates with the Workflow Orchestration capability for end-to-end processes and with Intelligent Document Processing for unstructured inputs. Modern RPA increasingly blends with AI to handle judgment-based steps, though API-led integration is preferred for durability.
