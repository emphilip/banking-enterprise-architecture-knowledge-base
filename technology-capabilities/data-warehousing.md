# Data Warehousing

**Definition.** Data Warehousing is the technology capability that consolidates structured and semi-structured data into governed, query-optimized stores supporting analytics, reporting, and historical analysis across the bank.
**Also known as:** Data Warehouse, Lakehouse.

- Data Warehousing mentions Databricks.
- Data Warehousing is related to Analytics Platform.

## Details
Data Warehousing provides the curated analytical layer consumed by reporting, risk, and machine-learning workloads. Legacy appliances such as Teradata, Oracle Exadata, and IBM Db2 are on-premise and capacity-constrained, while modern cloud platforms like Snowflake, Databricks, and Google BigQuery separate storage from compute and support the lakehouse pattern. It is fed by the Data Streaming and Integration Platform capabilities.
