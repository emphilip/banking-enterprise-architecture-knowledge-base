# Reject Inference Model

**Definition.** Reject Inference Model is the technology capability — a statistical or ML model — that infers likely performance of rejected or non-booked applicants to debias scorecards and widen the modelled population.
**Also known as:** Reject Inference Engine, Inference Scorecard.

## Relationships
- Reject Inference Model is defined in the Core Processing domain.
- Reject Inference Model is derived from Credit Scoring Service.
- Reject Inference Model depends on Machine Learning Platform.

## Details
Reject Inference Model addresses sample bias by inferring how rejected and non-booked applicants would have performed, so scorecards are built on a population that resembles the full through-the-door applicant base. Experian describes reject inference as a core scorecard-development technique that improves predictive accuracy and reduces bias when only the performance of accepted applicants is observed.
