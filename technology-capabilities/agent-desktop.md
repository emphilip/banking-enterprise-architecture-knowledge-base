# Agent Desktop

**Definition.** Agent Desktop is the technology sub-capability providing the unified agent workspace that presents interactions, customer context, guided workflows and AI assist across all channels in one interface.
**Also known as:** Agent Workspace, Unified Agent Desktop, Contact Centre Desktop.

## Relationships
- Agent Desktop is defined in the Channels & Engagement domain.
- Agent Desktop is derived from Contact Center Platform.
- Agent Desktop depends on Computer Telephony Integration.
- Agent Desktop is related to Agent Assist.

## Details
Agent Desktop consolidates call/chat controls, customer profile and interaction history, knowledge, case actions and real-time AI suggestions into a single pane so agents avoid swivel-chairing between systems. It receives screen-pop context on contact arrival from the telephony integration layer and surfaces next-best-action and live transcription prompts from AI assist. CCaaS platforms such as Amazon Connect and Genesys ship configurable agent workspaces that embed CRM and back-office apps directly in the workspace.
