# ISO 20022 Translation Service

**Definition.** ISO 20022 Translation Service is the technology capability providing dedicated bidirectional MT/MX translation, schema mapping and coexistence handling for the ISO 20022 standard during and after migration.
**Also known as:** MX Translation, MT-MX Mapping.

## Relationships
- ISO 20022 Translation Service is defined in the Core Processing domain.
- ISO 20022 Translation Service is derived from Payment Message Transformer.
- ISO 20022 Translation Service mentions ISO 20022.

## Details
ISO 20022 Translation Service performs bidirectional MT-to-MX and MX-to-MT translation, schema mapping and coexistence handling so legacy MT and new MX systems interoperate during the migration window. It preserves data integrity and flags truncation, a service offered by Volante ISO 20022 services and Bottomline message transformation.
