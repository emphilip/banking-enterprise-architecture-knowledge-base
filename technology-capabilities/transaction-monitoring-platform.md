# Transaction Monitoring Platform

**Definition.** Transaction Monitoring Platform is the technology capability that screens transactions and customer behaviour against AML typologies and scenarios to generate, triage, and investigate suspicious activity alerts.
**Also known as:** AML Platform.

- Transaction Monitoring Platform mentions Oracle Mantas.

- Transaction Monitoring Platform mentions Hawk AI.
- Transaction Monitoring Platform mentions ComplyAdvantage.

## Details
Transaction Monitoring Platform runs AML scenarios, network analysis, and alerting to support anti-money-laundering and suspicious activity reporting. Legacy systems such as Oracle Mantas and NICE Actimize are rule-heavy and generate high false-positive volumes, while modern platforms like Hawk AI and ComplyAdvantage apply machine learning to reduce noise and improve detection. It depends on the Machine Learning Platform and Data Streaming capabilities.
