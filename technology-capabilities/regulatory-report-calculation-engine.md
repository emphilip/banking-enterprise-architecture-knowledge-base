# Regulatory Report Calculation Engine

**Definition.** Regulatory Report Calculation Engine is the rules/calculation technology sub-capability that applies regulatory logic to a harmonised data model to compute report figures across jurisdictions and reporting areas.
**Also known as:** Report Calculation Engine, Reporting Computation Engine, Abacus Calculation Engine, Reg Calc Engine.

## Relationships
- Regulatory Report Calculation Engine is defined in the Data & Analytics domain.
- Regulatory Report Calculation Engine is derived from Regulatory Reporting Engine.
- Regulatory Report Calculation Engine depends on Regulatory Data Model.
- Regulatory Report Calculation Engine is related to Regulatory Reporting.

## Details
Regulatory Report Calculation Engine, exemplified by the Regnology Abacus calculation engine, applies regulatory rule sets — capital, liquidity, large exposures, statistical returns — to the harmonised regulatory data model to derive every reported figure. A single calculation core spanning regimes (Basel/CRR, COREP/FINREP, MAS, APRA and more) gives consistent, reproducible results, with full drill-down from a cell to its underlying granular records. Rule libraries are versioned by reporting reference date so historical and restated runs can be reproduced exactly.
