# Data Stewardship Workflow

**Definition.** Data Stewardship Workflow is the master data technology sub-capability providing a steward workbench with task queues, escalations and SLA-driven review for resolving potential matches, data-quality exceptions and manual overrides.
**Also known as:** Stewardship Console, Bulk Match Review.

## Relationships
- Data Stewardship Workflow is defined in the Data & Analytics domain.
- Data Stewardship Workflow is derived from Master Data Management.
- Data Stewardship Workflow depends on Workflow Orchestration.
- Data Stewardship Workflow is related to Data Governance.

## Details
Data Stewardship Workflow routes uncertain matches, data-quality exceptions and override requests to human stewards through prioritised queues with audit trails and SLA controls. It lets stewards confirm or reject suggested merges, edit golden records and bulk-review match candidates, closing the loop between automated entity resolution and governed manual intervention in master data platforms.
