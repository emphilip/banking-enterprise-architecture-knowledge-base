# Agent Assist

**Definition.** Agent Assist is the conversational AI technology sub-capability providing an AI co-pilot embedded in the agent desktop that transcribes, recommends next-best-action and drafts responses to support live human service agents.
**Also known as:** Agent Copilot, Next Best Action Assist.

## Relationships
- Agent Assist is defined in the AI & Automation domain.
- Agent Assist is derived from Conversational AI.
- Agent Assist depends on Generative AI Platform.
- Agent Assist is related to Contact Center.

## Details
Agent Assist works alongside human agents during live interactions, transcribing conversations in real time, suggesting answers and next-best actions, and drafting compliant responses and summaries. It draws on knowledge bases and generative AI to reduce handle time and improve consistency, and is delivered as a copilot inside contact-centre and CRM agent consoles.
