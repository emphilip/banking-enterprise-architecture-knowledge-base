# Intelligent Document Processing

**Definition.** Intelligent Document Processing is the technology capability that extracts, classifies, and validates data from unstructured documents using OCR, machine learning, and natural-language understanding.
**Also known as:** IDP.

## Relationships
- Intelligent Document Processing is defined in the AI & Automation domain.
- Intelligent Document Processing depends on the Machine Learning Platform capability.
- Intelligent Document Processing mentions Hyperscience.
- Intelligent Document Processing mentions AWS Textract.
- Intelligent Document Processing is related to Document Management.

## Details
Intelligent Document Processing automates data capture from KYC documents, loan files, and statements, replacing manual keying. Modern platforms such as Hyperscience and AWS Textract combine OCR with ML extraction and human-in-the-loop validation to reach high straight-through rates. It depends on the Machine Learning Platform capability and feeds data into the Document Management repository and downstream workflows.
