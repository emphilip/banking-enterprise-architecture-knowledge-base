# Least-Cost Routing Service

**Definition.** Least-Cost Routing Service is the technology capability providing cost-based routing logic that selects the cheapest viable rail/scheme meeting the payment's speed and cut-off requirements.
**Also known as:** Cost-Based Routing.

## Relationships
- Least-Cost Routing Service is defined in the Core Processing domain.
- Least-Cost Routing Service is derived from Payment Routing Engine.
- Least-Cost Routing Service is related to Scheme Connectivity Gateway.

## Details
Least-Cost Routing Service compares per-rail fees against each payment's speed and cut-off constraints to pick the cheapest viable scheme, optimising cost without breaching service-level requirements. It is a specialised routing function within payment hubs such as FIS Open Payment Framework and Volante VolPay Hub.
