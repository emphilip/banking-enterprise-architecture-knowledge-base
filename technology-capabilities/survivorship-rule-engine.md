# Survivorship Rule Engine

**Definition.** Survivorship Rule Engine is the master data technology sub-capability providing configurable field-level survivorship rules that select the prevailing attribute value across crosswalk records to build the golden record.
**Also known as:** Survivorship Rules, Field Survivorship.

## Relationships
- Survivorship Rule Engine is defined in the Data & Analytics domain.
- Survivorship Rule Engine is derived from Golden Record Management.
- Survivorship Rule Engine is related to Data Governance.

## Details
Survivorship Rule Engine evaluates configurable rules, such as most-recent, most-trusted-source or most-complete, to choose the winning value for each attribute when consolidating records into a golden party record. It maintains crosswalks to contributing sources for lineage and allows governance teams to tune rules per attribute, sitting at the heart of golden record construction in master data platforms.
