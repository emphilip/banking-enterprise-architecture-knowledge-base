# Card Processing

**Definition.** Card Processing is the technology capability that issues card credentials, authorizes transactions in real time, manages BINs and card lifecycle states, and settles interchange across Visa and Mastercard networks.
**Also known as:** Card Management System.

## Relationships
- Card Processing is defined in the Core Processing domain.
- Card Processing depends on the Core Banking Processing capability.
- Card Processing mentions TSYS TS2.
- Card Processing mentions First Data Cards.
- Card Processing mentions Marqeta.
- Card Processing mentions Galileo.
- Card Processing is related to Payment Orchestration.

## Details
Card Processing handles authorization, clearing, and settlement messaging with the schemes and maintains the card system of record. Legacy platforms such as TSYS TS2 and First Data Cards are large packaged systems, while modern processors like Marqeta and Galileo expose programmable, API-driven issuing for embedded and fintech card programs. It connects to fraud and dispute capabilities to manage card-not-present risk.
