# Network Entity Resolution

**Definition.** Network Entity Resolution is the technology sub-capability that resolves disparate records into single entities and builds relationship networks to provide context, uncover hidden links and prioritise financial-crime risk.
**Also known as:** Entity Resolution Engine, Network Analytics, Link Analysis.

## Relationships
- Network Entity Resolution is defined in the AI & Automation domain.
- Network Entity Resolution is derived from Transaction Monitoring Platform.
- Network Entity Resolution depends on Machine Learning Platform.
- Network Entity Resolution is related to Master Data Management.

## Details
Network Entity Resolution, as delivered by Quantexa contextual decision intelligence, matches records across internal and external sources into single resolved entities and constructs networks of relationships, transactions and shared attributes around them. This holistic context exposes hidden connections — shared addresses, directors, counterparties — that point-in-time alerts miss, sharpening risk prioritisation and reducing false positives. It complements scenario detection by giving investigators a network view of a subject, and overlaps the entity-resolution discipline of master data management.
