# Performance & Attribution Engine

**Definition.** Performance & Attribution Engine is the technology capability that calculates time- and money-weighted returns and decomposes performance via attribution against benchmarks, using holding-level data to explain contribution and risk across the total portfolio.
**Also known as:** Performance Measurement Engine, Attribution Analytics, Returns & Attribution Service.

## Relationships
- Performance & Attribution Engine is defined in the Core Processing domain.
- Performance & Attribution Engine is derived from Portfolio Management System.
- Performance & Attribution Engine depends on Portfolio Accounting Engine.
- Performance & Attribution Engine is related to Portfolio Management.

## Details
Performance & Attribution Engine computes time-weighted (TWR) and money-weighted (IRR) returns and runs Brinson-style attribution to split active return into allocation, selection and interaction effects against benchmarks. It produces GIPS-compliant composite and attribution reporting from the holding-level data sourced from the Portfolio Accounting Engine. Platforms such as SimCorp One and BlackRock Aladdin increasingly deliver real-time performance and attribution analytics, replacing end-of-day batch so portfolio managers can see contribution and risk intraday.
