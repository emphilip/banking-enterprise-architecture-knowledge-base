# Financial Data Quality Service

**Definition.** Financial Data Quality Service is the technology capability that loads, validates, maps and certifies source-system data before consolidation and planning, providing a built-in finance data integration and quality layer with full drill-back and audit.
**Also known as:** Financial Data Quality Management, FDQM, Data Integration & Validation (Finance).

## Relationships
- Financial Data Quality Service is defined in the Core Processing domain.
- Financial Data Quality Service is derived from Financial Consolidation Platform.
- Financial Data Quality Service is related to Consolidation Engine.

## Details
Financial Data Quality Service connects to source ledgers and ERPs, maps source accounts and entities to the consolidation/planning model, and runs validation and confirmation rules that must pass before data is locked for the close. Every consolidated figure can be drilled back through the mapping to the originating source record, giving auditors and controllers a transparent, certified data pipeline into the Consolidation Engine.
