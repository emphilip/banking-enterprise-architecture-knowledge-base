# Natural Language Query Service

**Definition.** Natural Language Query Service lets users ask questions of data in plain language and returns answers, charts, and automated insights.
**Also known as:** NL Query Service, Search Analytics, AI-Assisted Query.

## Relationships
- Natural Language Query Service is defined in the Data & Analytics domain.
- Natural Language Query Service is derived from Self-Service Analytics.
- Natural Language Query Service depends on Semantic Layer Service.

## Details
Natural Language Query Service interprets a typed or spoken question, maps it to entities and measures, and generates the underlying query, returning a result with an auto-chosen visualisation. Search-driven analytics tools such as ThoughtSpot pioneered this pattern, and modern BI platforms now layer large-language-model assistants on top. Accuracy and trust depend on grounding the model in a governed semantic layer: the curated metrics, synonyms, and permissions constrain interpretation so generated answers use approved definitions rather than guessing column meanings. This lowers the barrier to analytics for non-technical banking staff while keeping answers consistent with certified business logic.
