# Detection Scenario Library

**Definition.** Detection Scenario Library is the managed catalogue technology sub-capability holding out-of-the-box and custom AML/CTF scenarios, parameters and segmentation logic, with versioning and tuning of thresholds.
**Also known as:** Scenario Library, Scenario Manager, Rule Library.

## Relationships
- Detection Scenario Library is defined in the AI & Automation domain.
- Detection Scenario Library is derived from Scenario Detection Engine.
- Detection Scenario Library is related to Model Risk Management Platform.

## Details
Detection Scenario Library curates the inventory of monitoring scenarios that the Scenario Detection Engine executes: each scenario carries its detection logic, threshold sets per customer segment, effective dates and version history. Tuning workflows record above-the-line / below-the-line testing and threshold changes so that scenario performance is auditable and defensible to regulators. Because scenarios behave like models, their governance overlaps the model risk management discipline, including periodic effectiveness review and change control.
