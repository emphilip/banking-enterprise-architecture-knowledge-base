# Data Visualisation Library

**Definition.** Data Visualisation Library supplies the chart, graph, and geospatial rendering components and customisation options used to build visual analytics.
**Also known as:** Data Visualization Library, Charting Library, Visualisation Components.

## Relationships
- Data Visualisation Library is defined in the Data & Analytics domain.
- Data Visualisation Library is derived from Analytics Platform.
- Data Visualisation Library is related to Dashboarding Service.
- Data Visualisation Library is related to Embedded Analytics Service.

## Details
Data Visualisation Library provides the reusable building blocks of visual analytics: bar, line, scatter, heatmap, treemap, and geospatial chart types, plus axes, legends, tooltips, and theming controls that bind data fields to visual encodings. Libraries handle aggregation, scaling, and interactivity such as hover, zoom, and selection, and expose configuration so developers and dashboard authors can tailor appearance and behaviour. They are the rendering foundation that dashboarding and embedded analytics build upon, and mature toolkits such as Dundas Data Visualization offer rich customisation across many chart families. Consistent, accessible visual encodings improve how analysts and customers interpret banking data.
