# Omnichannel Routing Engine

**Definition.** Omnichannel Routing Engine is the technology sub-capability providing unified routing across voice, chat, email, messaging and tasks using skills, predictive and journey-aware logic to a single agent workforce.
**Also known as:** Omnichannel Routing, Interaction Routing Engine, Universal Queue.

## Relationships
- Omnichannel Routing Engine is defined in the Channels & Engagement domain.
- Omnichannel Routing Engine is derived from Contact Center Platform.
- Omnichannel Routing Engine is related to Automatic Call Distribution.

## Details
Omnichannel Routing Engine places voice, chat, email, social and async messaging plus back-office tasks into one universal queue and matches each to the best agent using skills, proficiency, priority, predictive (AI) routing and journey context. It blends concurrent digital interactions with calls under a single capacity model, preserving conversation continuity across channels. Genesys Cloud and Amazon Connect describe this engine as the cross-channel extension of ACD that load-balances a unified workforce.
