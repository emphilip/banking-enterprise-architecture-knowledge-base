# Financial Crime Case Manager

**Definition.** Financial Crime Case Manager is the investigation-workspace technology sub-capability that consolidates alerts and entity data into cases, managing workflow, audit trail, narrative and disposition for AML and financial-crime investigations.
**Also known as:** FinCrime Case Management, Investigation Workbench, Enterprise Case Management, ActOne.

## Relationships
- Financial Crime Case Manager is defined in the AI & Automation domain.
- Financial Crime Case Manager is derived from Transaction Monitoring Platform.
- Financial Crime Case Manager depends on Alert Scoring Service.
- Financial Crime Case Manager is related to Suspicious Activity Reporting.

## Details
Financial Crime Case Manager, exemplified by NICE Actimize ActOne and Oracle FCCM Enterprise Case Management, is the analyst workbench where prioritised alerts are grouped into investigations. It provides configurable workflows, four-eyes review, customer 360 and relationship views, document attachment and a structured narrative, all on an immutable audit trail. Dispositioned cases that warrant reporting hand off to suspicious-activity reporting and SAR/STR filing, and case outcomes feed back to tune detection and scoring.
