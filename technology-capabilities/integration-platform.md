# Integration Platform

**Definition.** Integration Platform is the technology capability that connects applications and data sources through messaging, transformation, routing, and connectors, enabling system-to-system interoperability across the bank.
**Also known as:** ESB, iPaaS.

- Integration Platform is related to API Management.

## Details
Integration Platform mediates between core systems, channels, and third parties using adapters, canonical models, and orchestration. Legacy enterprise service buses such as TIBCO BusinessWorks, webMethods, and Oracle SOA Suite are centralized and on-premise, while modern iPaaS like MuleSoft and Boomi are cloud-native with prebuilt connectors and API-led design. It underpins the API Management and Workflow Orchestration capabilities.
