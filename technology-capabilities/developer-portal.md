# Developer Portal

**Definition.** Developer Portal is the self-service hub where internal and partner developers discover, document, test, subscribe to, and obtain credentials for published APIs.
**Also known as:** API Developer Portal, API Consumer Portal, Self-Service API Portal.

## Relationships
- Developer Portal is defined in the Integration & APIs domain.
- Developer Portal is derived from API Management.
- Developer Portal depends on API Catalog & Registry.

## Details
Developer Portal renders OpenAPI/AsyncAPI specifications as interactive reference documentation, exposes a "try it" console that signs and replays live calls, and lets consumers self-register applications to receive subscription keys or OAuth client credentials. Azure API Management ships a customizable developer portal for onboarding, and IBM API Connect provides a portal for plan selection and usage dashboards. The portal reads its product and API listings from the platform catalog so published surface area stays consistent for consumers.
