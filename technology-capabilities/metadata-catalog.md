# Metadata Catalog

**Definition.** Metadata Catalog is the technology sub-capability providing a searchable catalogue of technical and business metadata that organises data assets, enables discovery and surfaces trusted, classified information.
**Also known as:** Data Catalog Service, Metadata Repository, Asset Catalog, Data Discovery.

## Relationships
- Metadata Catalog is defined in the Data & Analytics domain.
- Metadata Catalog is derived from Data Governance.
- Metadata Catalog depends on Data Governance.
- Metadata Catalog is related to Master Data Management.

## Details
Metadata Catalog, modelled on the Collibra Data Catalog, automatically harvests technical metadata from databases, files, BI tools and pipelines and enriches it with business context, ownership, classification and quality scores. Search, ratings and certification help analysts discover and trust the right datasets, while sensitivity tags drive access and privacy controls. It is the foundational registry that lineage, quality and stewardship capabilities reference, and it complements master data management by cataloguing where authoritative party and reference data live.
