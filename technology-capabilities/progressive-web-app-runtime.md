# Progressive Web App Runtime

**Definition.** Progressive Web App Runtime is the technology sub-capability providing installable, offline-capable progressive-web-app delivery of the web banking portal with service-worker caching and responsive layout.
**Also known as:** PWA Runtime, Responsive Web Runtime.

## Relationships
- Progressive Web App Runtime is defined in the Channels & Engagement domain.
- Progressive Web App Runtime is derived from Web Banking Portal.

## Details
Progressive Web App Runtime applies the PWA pattern to the web channel: a web app manifest enables add-to-home-screen install, a service worker caches the app shell for fast and resilient loads under poor connectivity, and the layout adapts responsively across form factors. It supports push notifications and background sync where the platform permits, narrowing the gap with native apps without an app-store download. As described in web.dev PWA guidance, this runtime lets a bank deliver an installable experience from the same responsive portal codebase.
