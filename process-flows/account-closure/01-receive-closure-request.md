# Receive Closure Request

**Definition.** Receive Closure Request captures an Account Closure Request from a customer or a dormancy trigger.
**Also known as:** Log Closure Request, Closure Intake Step.

## Relationships
- Receive Closure Request is defined in the Account Closure process.
- Receive Closure Request causes Validate Closure Eligibility.
- Receive Closure Request depends on the Digital Channel Platform capability.
- Receive Closure Request mentions Deposit Operations Clerk.
- Receive Closure Request mentions Account Closure Request.

## Details
The Deposit Operations Clerk logs the Account Closure Request from the channel or dormancy trigger. Controls include request logging and signatory capture.
