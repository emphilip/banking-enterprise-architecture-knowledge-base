# Implement Compliance Control

**Definition.** Implement Compliance Control implements the required control, policy and process changes to embed the new obligation.
**Also known as:** Embed Compliance Change, Control Implementation Step.

## Relationships
- Implement Compliance Control is defined in the Regulatory Change Management process.
- Implement Compliance Control causes Attest Change Compliance.
- Implement Compliance Control depends on the Policy & Control Management capability.
- Implement Compliance Control mentions Regulatory Change Manager.
- Implement Compliance Control mentions Regulatory Obligation.

## Details
The Regulatory Change Manager implements the remediation plan into implemented controls. Controls include control implementation, change governance and testing to embed the Regulatory Obligation.
