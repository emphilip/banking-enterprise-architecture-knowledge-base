# Validate Order

**Definition.** Validate Order validates the order and runs pre-trade compliance against mandate, limits and restricted lists.
**Also known as:** Order Validation Step, Pre-Trade Validation Step.

## Relationships
- Validate Order is defined in the Trade Order Management process.
- Validate Order causes Route Order.
- Validate Order depends on the Brokerage & Trading capability.
- Validate Order mentions Trader.
- Validate Order mentions Order Ticket.

## Details
The Trader validates the captured order and runs pre-trade compliance against mandate limits. Controls include pre-trade mandate/limit checks and restricted-list screening.
