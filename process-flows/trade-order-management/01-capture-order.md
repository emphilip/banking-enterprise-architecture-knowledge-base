# Capture Order

**Definition.** Capture Order captures the Order Ticket into the OMS with instrument, side, quantity and constraints.
**Also known as:** Order Capture Step, Intake Order Step.

## Relationships
- Capture Order is defined in the Trade Order Management process.
- Capture Order causes Validate Order.
- Capture Order depends on the Brokerage & Trading capability.
- Capture Order mentions Trader.
- Capture Order mentions Order Ticket.

## Details
The Trader captures the Order Ticket into the OMS. Controls include order completeness and mandate reference.
