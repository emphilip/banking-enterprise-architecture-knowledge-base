# Compile Regulatory Return

**Definition.** Compile Regulatory Return compiles the return against the prescribed template or taxonomy, such as FinCEN Form 112.
**Also known as:** Build Return, Return Compilation Step.

## Relationships
- Compile Regulatory Return is defined in the Regulatory Filing process.
- Compile Regulatory Return causes Validate Filing Quality.
- Compile Regulatory Return depends on the Regulatory Reporting Engine capability.
- Compile Regulatory Return mentions Regulatory Reporting Analyst.
- Compile Regulatory Return mentions CTR Filing.

## Details
The Regulatory Reporting Analyst compiles the aggregated dataset into a compiled return such as the CTR Filing on FinCEN Form 112. Controls include template/taxonomy conformance, mandatory fields and formatting.
