# Aggregate Filing Data

**Definition.** Aggregate Filing Data gathers and aggregates the underlying data from source systems for the return.
**Also known as:** Gather Filing Data, Data Aggregation Step (Filing).

## Relationships
- Aggregate Filing Data is defined in the Regulatory Filing process.
- Aggregate Filing Data causes Compile Regulatory Return.
- Aggregate Filing Data depends on the Data Warehousing capability.
- Aggregate Filing Data mentions Regulatory Reporting Analyst.
- Aggregate Filing Data mentions CTR Filing.

## Details
The Regulatory Reporting Analyst aggregates source data into an aggregated dataset for the return. Controls include data completeness, same-day aggregation for the CTR Filing and reconciliation to source.
