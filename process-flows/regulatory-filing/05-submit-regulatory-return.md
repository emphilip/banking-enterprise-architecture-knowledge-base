# Submit Regulatory Return

**Definition.** Submit Regulatory Return submits the validated return to the regulator via e-filing within the statutory deadline.
**Also known as:** File Return, Return Submission Step.

## Relationships
- Submit Regulatory Return is defined in the Regulatory Filing process.
- Submit Regulatory Return causes Archive Filing Record.
- Submit Regulatory Return depends on the Regulatory Reporting Engine capability.
- Submit Regulatory Return mentions Regulatory Reporting Analyst.
- Submit Regulatory Return mentions Regulatory Filing Submitted Event.

## Details
The Regulatory Reporting Analyst submits the validated return to produce a submitted return. Controls include e-filing, the CTR 15-day deadline, a no-extension control and submission integrity, with success emitting the Regulatory Filing Submitted Event.
