# Receive Maintenance Request

**Definition.** Receive Maintenance Request captures a Service Maintenance Request covering details, mandate, stop payment or standing instruction.
**Also known as:** Log Maintenance Request, Maintenance Intake Step.

## Relationships
- Receive Maintenance Request is defined in the Account Maintenance process.
- Receive Maintenance Request causes Verify Account Holder.
- Receive Maintenance Request depends on the Digital Channel Platform capability.
- Receive Maintenance Request mentions Deposit Operations Clerk.
- Receive Maintenance Request mentions Service Maintenance Request.

## Details
The Deposit Operations Clerk logs the Service Maintenance Request from the channel. Controls include request logging and channel authentication at intake.
