# Raise Rebalance Orders

**Definition.** Raise Rebalance Orders raises the resulting orders as Order Tickets into trading to execute the rebalance.
**Also known as:** Raise Orders Step, Order Generation Step.

## Relationships
- Raise Rebalance Orders is defined in the Portfolio Rebalancing process.
- Raise Rebalance Orders depends on the Brokerage & Trading capability.
- Raise Rebalance Orders mentions Portfolio Manager.
- Raise Rebalance Orders mentions Order Ticket.

## Details
The Portfolio Manager raises the Order Ticket set from the approved rebalance plan and hands off to Trade Order Management. Controls include order accuracy and the hand-off control to Trade Order Management.
