# Measure Portfolio Drift

**Definition.** Measure Portfolio Drift measures current holdings versus the Investment Policy Statement target allocation to quantify drift.
**Also known as:** Drift Measurement Step, Measure Allocation Drift.

## Relationships
- Measure Portfolio Drift is defined in the Portfolio Rebalancing process.
- Measure Portfolio Drift causes Evaluate Drift Threshold.
- Measure Portfolio Drift depends on the Portfolio Management capability.
- Measure Portfolio Drift mentions Portfolio Manager.
- Measure Portfolio Drift mentions Investment Policy Statement.

## Details
The Portfolio Manager quantifies drift from holdings and valuations against the Investment Policy Statement targets. Controls include IPS targets and valuation accuracy.
