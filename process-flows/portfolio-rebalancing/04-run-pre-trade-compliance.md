# Run Pre-Trade Compliance

**Definition.** Run Pre-Trade Compliance pre-trade compliance-checks the rebalance proposal against mandate and IPS limits and restricted lists.
**Also known as:** Pre-Trade Compliance Step, Compliance Screening Step.

## Relationships
- Run Pre-Trade Compliance is defined in the Portfolio Rebalancing process.
- Run Pre-Trade Compliance causes Approve Rebalance Plan.
- Run Pre-Trade Compliance depends on the Portfolio Management capability.
- Run Pre-Trade Compliance mentions Portfolio Manager.

## Details
The Portfolio Manager runs pre-trade compliance on the rebalance proposal against mandate limits. Controls include pre-trade compliance, mandate/IPS limits, and restricted-list screening.
