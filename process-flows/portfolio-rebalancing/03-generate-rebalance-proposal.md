# Generate Rebalance Proposal

**Definition.** Generate Rebalance Proposal generates a rebalance proposal of buys and sells to restore the target, considering tax lots and costs.
**Also known as:** Rebalance Proposal Step, Propose Rebalance Step.

## Relationships
- Generate Rebalance Proposal is defined in the Portfolio Rebalancing process.
- Generate Rebalance Proposal causes Run Pre-Trade Compliance.
- Generate Rebalance Proposal depends on the Portfolio Management capability.
- Generate Rebalance Proposal mentions Portfolio Manager.

## Details
The Portfolio Manager generates the buy/sell rebalance proposal from the rebalance decision and holdings. Controls include tax-lot and wash-sale checks and cost awareness.
