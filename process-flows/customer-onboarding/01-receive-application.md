# Receive Application

**Definition.** Receive Application captures a prospect's onboarding application from a digital channel, branch or API and opens an onboarding case.
**Also known as:** Capture Application, Intake Applicant.

## Relationships
- Receive Application is defined in the Customer Onboarding process.
- Receive Application causes Capture Consent.
- Receive Application depends on the Digital Channel Platform capability.
- Receive Application mentions Onboarding Specialist.
- Receive Application mentions Customer Application.
- Receive Application mentions Onboarding Case.
- Receive Application mentions Application Submitted Event.

## Details
The Onboarding Specialist captures the application. Inputs are the Customer Application; outputs are an Onboarding Case. The Application Submitted Event triggers the flow. Controls include channel authentication and intake logging.
