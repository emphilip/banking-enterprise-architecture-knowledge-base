# Resolve Self-Service Request

**Definition.** Resolve Self-Service Request resolves the request in-channel via self-service, executing the servicing action.
**Also known as:** In-Channel Resolution, Self-Serve Action.

## Relationships
- Resolve Self-Service Request is defined in the Channel Servicing process.
- Resolve Self-Service Request causes Escalate To Assisted Servicing.
- Resolve Self-Service Request depends on the Core Banking Processing capability.
- Resolve Self-Service Request mentions Channel Operations Analyst.

## Details
The Channel Operations Analyst supports in-channel resolution. Inputs are the orchestrated journey; outputs are a resolved request. Controls include action authorisation and accessibility.
