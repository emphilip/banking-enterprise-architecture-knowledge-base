# Start Digital Application

**Definition.** Start Digital Application opens the digital onboarding journey, presents product selection and begins a Digital Application with save-and-resume.
**Also known as:** Begin Online Application, Start DAO.

## Relationships
- Start Digital Application is defined in the Digital Onboarding Journey process.
- Start Digital Application causes Capture Applicant Details.
- Start Digital Application depends on the Digital Banking capability.
- Start Digital Application mentions Digital Onboarding Specialist.
- Start Digital Application mentions Digital Application.

## Details
The Digital Onboarding Specialist owns the journey start. Inputs are the product selection; outputs are a started Digital Application and a Channel Session. Controls include consent capture, accessibility (WCAG) and abandonment / save-and-resume; eKYC is deferred to the identity step.
