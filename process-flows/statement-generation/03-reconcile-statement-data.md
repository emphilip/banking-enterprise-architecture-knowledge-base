# Reconcile Statement Data

**Definition.** Reconcile Statement Data reconciles extracted activity to ledger control totals before rendering.
**Also known as:** Validate Statement Totals, Statement Recon Step.

## Relationships
- Reconcile Statement Data is defined in the Statement Generation process.
- Reconcile Statement Data causes Assemble Statement Document.
- Reconcile Statement Data depends on the Data Warehousing capability.
- Reconcile Statement Data mentions Reconciliation Analyst.

## Details
The Reconciliation Analyst reconciles extracted activity to control totals and produces a reconciled dataset. Controls include control-total reconciliation prior to statement assembly.
