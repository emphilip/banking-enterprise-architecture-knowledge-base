# Render Statement Output

**Definition.** Render Statement Output renders print, PDF and e-statement formats per the delivery preference.
**Also known as:** Produce Statement Output, Rendering Step.

## Relationships
- Render Statement Output is defined in the Statement Generation process.
- Render Statement Output causes Deliver Statement.
- Render Statement Output depends on the Document Management capability.
- Render Statement Output mentions Statement Production Officer.
- Render Statement Output mentions Bank Account Statement.

## Details
The Statement Production Officer renders the assembled statement into a Bank Account Statement per delivery preferences. Controls include format integrity and e-delivery consent.
