# Compile KYC Profile

**Definition.** Compile KYC Profile compiles the verified identity, screening results and risk indicators into the KYC Profile.
**Also known as:** Build KYC Profile, Profile Compilation Step.

## Relationships
- Compile KYC Profile is defined in the KYC Verification process.
- Compile KYC Profile causes Approve KYC Outcome.
- Compile KYC Profile depends on the KYC Management capability.
- Compile KYC Profile mentions KYC Analyst.
- Compile KYC Profile mentions KYC Profile.

## Details
The KYC Analyst consolidates the verified identity and screening results into the KYC Profile. Controls include profile completeness, mandatory fields and data quality so downstream approval has a complete record.
