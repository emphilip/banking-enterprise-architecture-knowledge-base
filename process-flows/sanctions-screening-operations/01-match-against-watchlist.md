# Match Against Watchlist

**Definition.** Match Against Watchlist screens party and transaction data against OFAC/UN/EU lists using exact and fuzzy matching.
**Also known as:** Run Watchlist Match, Watchlist Matching Step.

## Relationships
- Match Against Watchlist is defined in the Sanctions Screening Operations process.
- Match Against Watchlist causes Generate Sanctions Alert.
- Match Against Watchlist depends on the Sanctions Screening capability.
- Match Against Watchlist mentions Compliance Screening Officer.
- Match Against Watchlist mentions Sanctions Hit.

## Details
The Compliance Screening Officer screens party and transaction data against current lists to produce candidate matches. Controls include matching thresholds, fuzzy logic, list currency and the OFAC 50% rule for entities owned by sanctioned parties.
