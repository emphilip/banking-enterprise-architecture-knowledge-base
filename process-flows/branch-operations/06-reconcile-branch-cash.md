# Reconcile Branch Cash

**Definition.** Reconcile Branch Cash consolidates teller and vault positions, reconciles the branch cash GL and returns cash to the vault under dual custody.
**Also known as:** Branch Balancing, EOD Recon.

## Relationships
- Reconcile Branch Cash is defined in the Branch Operations process.
- Reconcile Branch Cash causes Close Branch.
- Reconcile Branch Cash depends on the Branch Banking capability.
- Reconcile Branch Cash mentions Branch Manager.

## Details
The Branch Manager reconciles the branch. Inputs are reconciled drawers and vault cash; outputs are a balanced branch and a reconciliation report. Branch balancing controls apply: dual custody, branch cash GL reconciliation, and retention of records.
