# Open Branch Cash

**Definition.** Open Branch Cash opens the branch by drawing teller floats from the vault under dual custody and recording opening balances.
**Also known as:** Branch Opening, Float Issuance.

## Relationships
- Open Branch Cash is defined in the Branch Operations process.
- Open Branch Cash causes Serve Counter Customer.
- Open Branch Cash depends on the Branch Banking capability.
- Open Branch Cash mentions Branch Manager.

## Details
The Branch Manager opens the branch. Inputs are vault cash and opening balances; outputs are issued teller floats. Branch balancing controls apply: dual custody of the vault and opening-balance sign-off govern float issuance.
