# Close Branch

**Definition.** Close Branch secures the vault and premises, finalises end-of-day controls and signs off branch closure.
**Also known as:** Branch Closing, EOD Close.

## Relationships
- Close Branch is defined in the Branch Operations process.
- Close Branch depends on the Branch Banking capability.
- Close Branch mentions Branch Manager.

## Details
The Branch Manager closes the branch. Inputs are the reconciliation report; outputs are a closed branch. Controls include vault securing and closing sign-off, completing the branch balancing cycle.
