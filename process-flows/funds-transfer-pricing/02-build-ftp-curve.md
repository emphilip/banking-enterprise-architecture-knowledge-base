# Build FTP Curve

**Definition.** Build FTP Curve builds the matched-maturity FTP curve and adds liquidity-premium and contingency add-ons by tenor.

## Relationships
- Build FTP Curve is defined in the Funds Transfer Pricing process.
- Build FTP Curve causes Set Behavioural Assumptions.
- Build FTP Curve depends on the Asset Liability Management capability.
- Build FTP Curve mentions ALM Analyst.
- Build FTP Curve mentions FTP Curve.

## Details
The ALM Analyst builds the matched-maturity FTP Curve and adds liquidity-premium and contingency add-ons by tenor. Inputs are the curated market curves and premium parameters; outputs are the draft FTP Curve. Controls include matched-maturity methodology governance and premium-range limits.
