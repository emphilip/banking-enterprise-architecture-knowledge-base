# Set Behavioural Assumptions

**Definition.** Set Behavioural Assumptions sets behavioural assumptions (non-maturity deposit tenor, prepayment) used to assign transfer tenors.

## Relationships
- Set Behavioural Assumptions is defined in the Funds Transfer Pricing process.
- Set Behavioural Assumptions causes Approve FTP Methodology.
- Set Behavioural Assumptions depends on the Asset Liability Management capability.
- Set Behavioural Assumptions mentions ALM Analyst.

## Details
The ALM Analyst sets behavioural assumptions for non-maturity deposit tenor and prepayment used to assign transfer tenors. Inputs are behavioural studies and product data; outputs are the behavioural assumption set. Controls include behavioural-assumption review and model documentation.
