# Log Complaint

**Definition.** Log Complaint records an inbound complaint from any channel into the case system with a reference number and key details.
**Also known as:** Register Complaint, Complaint Logging Step.

## Relationships
- Log Complaint is defined in the Complaint Handling process.
- Log Complaint causes Acknowledge Complaint.
- Log Complaint depends on the Case Management capability.
- Log Complaint mentions Complaints Officer.
- Log Complaint mentions Complaint Record.
- Log Complaint mentions Complaint Received Event.

## Details
The Complaints Officer records the complaint. The Complaint Received Event triggers the flow. Inputs are the Complaint Record; outputs are a logged complaint. Controls include mandatory-fields capture, reference issuance and record retention.
