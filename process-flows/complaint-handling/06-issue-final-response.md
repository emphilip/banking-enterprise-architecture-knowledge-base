# Issue Final Response

**Definition.** Issue Final Response sends the final response with the outcome and, where applicable, Ombudsman referral rights within the deadline.
**Also known as:** Send Final Response, Final Response Step.

## Relationships
- Issue Final Response is defined in the Complaint Handling process.
- Issue Final Response causes Conduct Root Cause.
- Issue Final Response depends on the Notification Services capability.
- Issue Final Response mentions Complaints Officer.
- Issue Final Response mentions Final Response Letter.

## Details
The Complaints Officer issues the Final Response Letter. Inputs are the redress decision; outputs are the Final Response Letter. Controls include the 8-week final-response SLA and FOS/escalation referral rights.
