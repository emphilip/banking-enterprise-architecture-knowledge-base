# Take Mortgage Application

**Definition.** Take Mortgage Application captures the URLA/Form 1003 application and authorizes the credit pull.
**Also known as:** Capture 1003, Mortgage Intake Step.

## Relationships
- Take Mortgage Application is defined in the Mortgage Origination process.
- Take Mortgage Application causes Issue Loan Estimate.
- Take Mortgage Application depends on the Loan Origination Platform capability.
- Take Mortgage Application mentions Mortgage Loan Originator.

## Details
The Mortgage Loan Originator takes the URLA/Form 1003. Inputs are the Mortgage Application; outputs are a captured mortgage file. Controls include URLA completeness, an application-date control and KYC capture. The Loan Application Submitted Event triggers the workflow.
