# Order Property Appraisal

**Definition.** Order Property Appraisal orders an independent appraisal to establish collateral market value.
**Also known as:** Order Valuation, Appraisal Step.

## Relationships
- Order Property Appraisal is defined in the Mortgage Origination process.
- Order Property Appraisal causes Review Title And Liens.
- Order Property Appraisal depends on the Collateral Management capability.
- Order Property Appraisal mentions Loan Processor.
- Order Property Appraisal mentions Collateral Appraisal.

## Details
The Loan Processor orders the appraisal. Inputs are the Loan Estimate; outputs are a Collateral Appraisal. Controls include appraiser independence, valuation standards and appraisal disclosure.
