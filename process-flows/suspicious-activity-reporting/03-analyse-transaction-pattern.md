# Analyse Transaction Pattern

**Definition.** Analyse Transaction Pattern analyses transactions and source of funds for patterns and anomalies indicating money laundering.
**Also known as:** Examine Transactions, Pattern Analysis Step.

## Relationships
- Analyse Transaction Pattern is defined in the Suspicious Activity Reporting process.
- Analyse Transaction Pattern causes Determine Suspicion.
- Analyse Transaction Pattern depends on the Transaction Monitoring capability.
- Analyse Transaction Pattern mentions AML Investigator.
- Analyse Transaction Pattern mentions Suspicious Activity Detected Event.

## Details
The AML Investigator examines the investigation case and transaction history to produce investigation findings. Controls include transaction analysis, source-of-funds checks and typology mapping, building the evidence that may raise a Suspicious Activity Detected Event.
