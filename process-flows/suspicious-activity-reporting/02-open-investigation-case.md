# Open Investigation Case

**Definition.** Open Investigation Case opens a case in the case-management system and assembles customer and transaction context.
**Also known as:** Create AML Case.

## Relationships
- Open Investigation Case is defined in the Suspicious Activity Reporting process.
- Open Investigation Case causes Analyse Transaction Pattern.
- Open Investigation Case depends on the Transaction Monitoring Platform capability.
- Open Investigation Case mentions AML Investigator.
- Open Investigation Case mentions KYC Profile.

## Details
The AML Investigator opens an investigation case from triaged alerts, pulling the KYC Profile and transaction context. Controls include case logging, context assembly and audit trail.
