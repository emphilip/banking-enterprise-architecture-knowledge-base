# Capture Investor Profile

**Definition.** Capture Investor Profile captures investor identity, financial situation and objectives to start the wealth relationship.
**Also known as:** Capture Investor Data, Investor Intake Step.

## Relationships
- Capture Investor Profile is defined in the Wealth Onboarding process.
- Capture Investor Profile causes Capture Risk Profile.
- Capture Investor Profile depends on the Investment Advisory capability.
- Capture Investor Profile mentions Wealth Relationship Manager.
- Capture Investor Profile mentions KYC Profile.

## Details
The Wealth Relationship Manager captures the investor profile from client data and the KYC Profile. Controls include KYC for investors and data completeness.
