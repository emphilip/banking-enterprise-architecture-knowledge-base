# Provision Wallet Token

**Definition.** Provision Wallet Token tokenizes the PAN and provisions the card to digital wallets.

## Relationships
- Provision Wallet Token is defined in the Card Issuance process.
- Provision Wallet Token depends on the Card Processing capability.
- Provision Wallet Token mentions Card Credential.
- Provision Wallet Token mentions Card Issuance Officer.

## Details
Provision Wallet Token network-tokenizes the PAN and provisions the credential to digital wallets, enforcing tokenization and wallet-provisioning controls. This is the final step of Card Issuance.
