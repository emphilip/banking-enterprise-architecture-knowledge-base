# File Suspicious Activity Report

**Definition.** File Suspicious Activity Report is the decision point where, when thresholds are met, a SAR is filed within regulatory deadlines.
**Also known as:** File SAR, Suspicious Activity Filing Step.

## Relationships
- File Suspicious Activity Report is defined in the Fraud Investigation process.
- File Suspicious Activity Report causes Feed Detection Models.
- File Suspicious Activity Report depends on the Transaction Monitoring Platform capability.
- File Suspicious Activity Report mentions Fraud Investigator.
- File Suspicious Activity Report mentions Fraud Case.

## Details
The Fraud Investigator branches on the SAR thresholds, filing from the fraud disposition and Fraud Case. Controls include the SAR $5,000/$25,000 thresholds, the 30/60-day filing window and narrative quality.
