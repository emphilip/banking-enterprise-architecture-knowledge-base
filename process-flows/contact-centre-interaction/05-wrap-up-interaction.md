# Wrap Up Interaction

**Definition.** Wrap Up Interaction performs after-call work: disposition, notes and create or update of the Interaction Record.
**Also known as:** After-Call Work, Disposition & Notes.

## Relationships
- Wrap Up Interaction is defined in the Contact Centre Interaction process.
- Wrap Up Interaction causes Score Interaction Quality.
- Wrap Up Interaction depends on the CRM Platform capability.
- Wrap Up Interaction mentions Contact Centre Agent.
- Wrap Up Interaction mentions Interaction Record.

## Details
The Contact Centre Agent completes after-call work. Inputs are the resolved interaction; outputs are an Interaction Record and a disposition. Controls include disposition accuracy and after-call work standards.
