# Check Available Limit

**Definition.** Check Available Limit checks the available credit limit or funds against the requested amount.

## Relationships
- Check Available Limit is defined in the Card Transaction Authorization process.
- Check Available Limit causes Apply Velocity Rules.
- Check Available Limit depends on the Card Processing capability.
- Check Available Limit mentions Card Operations Analyst.

## Details
Check Available Limit compares the requested amount to the open-to-buy credit limit or available funds, enforcing the credit-limit / available-funds check.
