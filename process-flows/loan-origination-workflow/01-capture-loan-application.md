# Capture Loan Application

**Definition.** Capture Loan Application captures the borrower's loan request and personal and financial data into the origination platform.
**Also known as:** Intake Loan Request, Loan Capture Step.

## Relationships
- Capture Loan Application is defined in the Loan Origination Workflow process.
- Capture Loan Application causes Pre-Qualify Borrower.
- Capture Loan Application depends on the Loan Origination Platform capability.
- Capture Loan Application mentions Loan Officer.
- Capture Loan Application mentions Loan Application.

## Details
The Loan Officer captures the request. Inputs are the Loan Application; outputs are a captured loan file. Controls include KYC identity capture, consent to the credit pull and data-quality validation. The Loan Application Submitted Event triggers the workflow.
