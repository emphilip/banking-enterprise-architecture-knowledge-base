# Close Request

**Definition.** Close Request sets the ticket to closed, captures feedback and retains the record, reopening if the customer responds within the window.
**Also known as:** Resolve & Close Request, Request Closure Step.

## Relationships
- Close Request is defined in the Service Request Handling process.
- Close Request depends on the Case Management capability.
- Close Request mentions Service Advisor.
- Close Request mentions Service Request Ticket.

## Details
The Service Advisor closes the Service Request Ticket. Inputs are the confirmed resolution; outputs are a closed request. Controls include closure control, feedback capture and a reopen window branch. This is the final step of the Service Request Handling flow.
