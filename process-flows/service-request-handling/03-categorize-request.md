# Categorize Request

**Definition.** Categorize Request classifies the request type, assigns priority and SLA and routes to the appropriate fulfilment queue or specialist.
**Also known as:** Classify Request, Request Triage Step.

## Relationships
- Categorize Request is defined in the Service Request Handling process.
- Categorize Request causes Assess Request.
- Categorize Request depends on the Workflow Orchestration capability.
- Categorize Request mentions Service Advisor.
- Categorize Request mentions Service Request Ticket.

## Details
The Service Advisor classifies and routes the request. Inputs are the verified requester; outputs are a triaged request. The priority/SLA assignment and routing rules branch across self-serve, agent and specialist.
