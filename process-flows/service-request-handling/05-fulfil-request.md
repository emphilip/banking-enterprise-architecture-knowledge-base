# Fulfil Request

**Definition.** Fulfil Request executes the requested change or service in the system of record, such as updating details, issuing a document or amending a product.
**Also known as:** Action Request, Request Fulfilment Step.

## Relationships
- Fulfil Request is defined in the Service Request Handling process.
- Fulfil Request causes Confirm Resolution.
- Fulfil Request depends on the Core Banking Processing capability.
- Fulfil Request mentions Service Advisor.
- Fulfil Request mentions Service Request Ticket.

## Details
The Service Advisor executes the change in the system of record. Inputs are the assessed request; outputs are a fulfilled request. Controls include change control and four-eyes for sensitive changes.
