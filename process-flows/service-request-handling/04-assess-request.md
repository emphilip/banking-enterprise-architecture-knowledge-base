# Assess Request

**Definition.** Assess Request determines whether the request can be fulfilled directly, needs approval, or must convert or escalate, for example to a complaint.
**Also known as:** Evaluate Request, Request Assessment Step.

## Relationships
- Assess Request is defined in the Service Request Handling process.
- Assess Request causes Fulfil Request.
- Assess Request depends on the Case Management capability.
- Assess Request mentions Service Advisor.
- Assess Request mentions Service Request Ticket.

## Details
The Service Advisor evaluates the request. Inputs are the triaged request; outputs are an assessed request. The eligibility/approval check branches across fulfil, escalate and convert to complaint.
