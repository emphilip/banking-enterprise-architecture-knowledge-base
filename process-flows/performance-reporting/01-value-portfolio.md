# Value Portfolio

**Definition.** Value Portfolio values all holdings and cash at period end using market prices.
**Also known as:** Portfolio Valuation Step, Mark-to-Market Step.

## Relationships
- Value Portfolio is defined in the Performance Reporting process.
- Value Portfolio causes Process Corporate Actions.
- Value Portfolio depends on the Portfolio Management capability.
- Value Portfolio mentions Performance Analyst.

## Details
The Performance Analyst values holdings and cash at period end from market prices. Controls include valuation accuracy and pricing-source control.
