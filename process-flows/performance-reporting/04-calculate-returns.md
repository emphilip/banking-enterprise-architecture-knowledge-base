# Calculate Returns

**Definition.** Calculate Returns computes time-weighted returns for the period and since inception.
**Also known as:** Return Calculation Step, TWR Calculation Step.

## Relationships
- Calculate Returns is defined in the Performance Reporting process.
- Calculate Returns causes Compute Attribution.
- Calculate Returns depends on the Portfolio Management capability.
- Calculate Returns mentions Performance Analyst.

## Details
The Performance Analyst computes time-weighted returns from reconciled positions and cash flows. Controls include GIPS time-weighted return methodology and cash-flow handling.
