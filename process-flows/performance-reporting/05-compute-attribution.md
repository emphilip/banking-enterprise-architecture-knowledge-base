# Compute Attribution

**Definition.** Compute Attribution compares returns to the benchmark and decomposes excess return into allocation and selection effects.
**Also known as:** Attribution Analysis Step, Benchmark Comparison Step.

## Relationships
- Compute Attribution is defined in the Performance Reporting process.
- Compute Attribution causes Produce Performance Report.
- Compute Attribution depends on the Portfolio Management capability.
- Compute Attribution mentions Performance Analyst.

## Details
The Performance Analyst decomposes excess return into allocation and selection effects against benchmark data. Controls include benchmark consistency and attribution methodology.
