# Produce Performance Report

**Definition.** Produce Performance Report assembles, signs off and delivers the client Performance Report with returns, attribution and disclosures.
**Also known as:** Report Production Step, Report Delivery Step.

## Relationships
- Produce Performance Report is defined in the Performance Reporting process.
- Produce Performance Report depends on the Portfolio Management capability.
- Produce Performance Report mentions Performance Analyst.
- Produce Performance Report mentions Performance Report.

## Details
The Performance Analyst assembles and delivers the Performance Report from the return figures and attribution analysis. Controls include disclosure accuracy, four-eyes sign-off, and GIPS disclosures.
