# Decide Pay Or Return

**Definition.** Decide Pay Or Return decides to pay the item as an overdraft or return it as NSF.
**Also known as:** Make Pay Return Decision, Pay/Return Decision Step.

## Relationships
- Decide Pay Or Return is defined in the Overdraft Servicing process.
- Decide Pay Or Return causes Apply Overdraft Fee.
- Decide Pay Or Return depends on the Decision Rules Engine capability.
- Decide Pay Or Return mentions Overdraft Operations Analyst.

## Details
The Overdraft Operations Analyst decides pay or return from the limit decision input. Controls include the pay/return policy and decision audit logging.
