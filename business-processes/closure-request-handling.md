# Closure Request Handling

**Definition.** Closure Request Handling is the capture and validation of a closure request and the confirmation of eligibility against holds, pending items and signatories.
**Also known as:** Closure Intake, Closure Validation.

## Relationships
- Closure Request Handling is derived from Account Closure.
- Closure Request Handling is defined in the Deposits & Accounts domain.
- Closure Request Handling depends on the Account Servicing capability.

## Details
Closure Request Handling takes the Account Closure Request and produces an eligible closure. Controls include signatory verification and hold checks.
