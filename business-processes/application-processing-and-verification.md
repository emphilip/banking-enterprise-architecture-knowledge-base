# Application Processing & Verification

**Definition.** Application Processing & Verification verifies income, employment and assets and completes the file so it can be decisioned and funded.
**Also known as:** Loan Processing, File Verification.

## Relationships
- Application Processing & Verification is derived from Loan Origination Workflow.
- Application Processing & Verification is defined in the Lending & Credit domain.
- Application Processing & Verification depends on the Document Management capability.

## Details
The sub-process is performed by the Loan Processor. Inputs are the pre-qualified application; outputs are a decision-ready file. Controls include income and employment verification, document completeness and AML screening.
