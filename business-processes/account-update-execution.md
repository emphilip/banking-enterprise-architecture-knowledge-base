# Account Update Execution

**Definition.** Account Update Execution is the application of requested changes to details, mandate, stop payment or standing instruction, and the posting of interest and fee changes with notice.
**Also known as:** Update Execution, Servicing Action.

## Relationships
- Account Update Execution is derived from Account Maintenance.
- Account Update Execution is defined in the Deposits & Accounts domain.
- Account Update Execution depends on the Core Banking Processing capability.

## Details
Account Update Execution takes a verified request and produces an updated account record. Controls include dual control and the Reg DD change-in-terms notice.
