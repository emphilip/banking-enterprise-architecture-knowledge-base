# SWIFT Message Processing

**Definition.** SWIFT Message Processing formats, validates and exchanges cross-border payment messages over SWIFT using ISO 20022 (CBPR+ pacs/pain) and MT formats, supporting the BIAN Correspondent Bank Operations and Payment Execution service domains.
**Also known as:** SWIFT Messaging, CBPR+, MT/MX.

## Relationships
- SWIFT Message Processing is defined in the Payments domain.
- SWIFT Message Processing is derived from Cross-Border Payments.
- SWIFT Message Processing depends on the Integration Platform capability.

## Details
SWIFT Message Processing grounds in the BIAN Correspondent Bank Operations and Payment Execution service domains. It formats, validates and exchanges cross-border payment messages over SWIFT using ISO 20022 CBPR+ pacs/pain and legacy MT formats. It manages MT/MX coexistence during migration.
