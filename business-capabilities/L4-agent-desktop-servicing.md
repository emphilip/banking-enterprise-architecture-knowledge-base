# Agent Desktop Servicing

**Definition.** Agent Desktop Servicing is the Channels & Engagement capability that equips agents with a unified workbench of customer context and servicing actions during contacts, realising the BIAN Customer Workbench service domain; it is named to avoid the Agent Assist technology sub-capability.
**Also known as:** Agent Workbench, Servicing Agent Desktop.

## Relationships
- Agent Desktop Servicing is defined in the Channels & Engagement domain.
- Agent Desktop Servicing is derived from Inbound Contact Servicing.
- Agent Desktop Servicing depends on the CRM Platform capability.

## Details
Agent Desktop Servicing realises the BIAN Customer Workbench: a single pane that assembles identity, accounts, recent interactions and next-best-action so the agent avoids swivel-chairing across systems. It surfaces omnichannel context from Channel Activity History so an agent sees the in-app journey a customer abandoned before calling, cutting handle time and repeat authentication. Role- and entitlement-based access enforces least-privilege over customer data under GDPR, and screen-level masking protects card and SSN fields. Modern desktops embed AI agent-assist (real-time knowledge surfacing, call summarisation) while keeping the human accountable for regulated decisions.
