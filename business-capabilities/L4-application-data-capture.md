# Application Data Capture

**Definition.** Application Data Capture is the capability that collects and validates borrower, income and asset data for credit applications, mapping to the BIAN Consumer Loan and Corporate Loan service domains.
**Also known as:** Loan Data Intake, Applicant Data Capture.

## Relationships
- Application Data Capture is defined in the Lending & Credit domain.
- Application Data Capture is derived from Loan Application Management.
- Application Data Capture depends on the Loan Origination Platform capability.
- Application Data Capture depends on the Document Management capability.

## Details
The BIAN Consumer Loan and Corporate Loan service domains require a complete applicant data set, which Application Data Capture assembles and validates. A concrete banking specific: a US residential mortgage application is captured on the standardised Uniform Residential Loan Application (URLA / Fannie Mae Form 1003), covering borrower, employment, income, assets and declarations.
