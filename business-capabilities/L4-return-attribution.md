# Return Attribution

**Definition.** Return Attribution decomposes portfolio returns into allocation, selection, and interaction effects versus a benchmark; it informs the BIAN Investment Portfolio Analysis service domain.
**Also known as:** Performance Attribution, Brinson Attribution.

## Relationships
- Return Attribution is defined in the Wealth & Investments domain.
- Return Attribution is derived from Performance Measurement.
- Return Attribution depends on the Portfolio Management System capability.
- Return Attribution depends on the Analytics Platform capability.

## Details
Return Attribution applies the Brinson-Hood-Beebower model to split active return into an allocation effect (over- or under-weighting sectors versus benchmark), a selection effect (choosing securities within a sector), and an interaction effect. This explains where the manager added or lost value relative to the benchmark and feeds GIPS-aligned client reporting and the BIAN Investment Portfolio Analysis service domain.
