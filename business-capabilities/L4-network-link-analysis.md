# Network Link Analysis

**Definition.** Network Link Analysis surfaces hidden relationships across parties, accounts and transactions to evidence suspicious networks during investigation, supporting FATF typology analysis and SAR decisions.
**Also known as:** Entity Network Analysis, Link Analytics.

## Relationships
- Network Link Analysis is defined in the Compliance & Financial Crime domain.
- Network Link Analysis is derived from AML Case Investigation.
- Network Link Analysis depends on the Analytics Platform capability.
- Network Link Analysis depends on the Machine Learning Platform capability.

## Details
FATF typology reports show that laundering and terrorist financing typically operate through networks of mules, shell companies and intermediary accounts rather than isolated transactions. Network Link Analysis builds graphs linking parties by shared identifiers (address, phone, device, beneficial owner) and fund flows to reveal funnel accounts, smurfing rings and circular transfers that single-account monitoring misses. The evidenced network enriches the SAR narrative and informs whether continuing-activity reviews and additional filings are warranted.
