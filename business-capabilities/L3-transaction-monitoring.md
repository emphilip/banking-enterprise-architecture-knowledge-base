# Transaction Monitoring

**Definition.** Transaction Monitoring is the business capability that screens transactions in real time and in batch against money-laundering typologies and behavioral baselines to generate alerts. It corresponds to the BIAN Financial Crime transaction surveillance service domain.
**Also known as:** Transaction Surveillance.

## Relationships
- Transaction Monitoring is defined in the Compliance & Financial Crime domain.
- Transaction Monitoring is derived from AML Monitoring.
- Transaction Monitoring depends on the Transaction Monitoring Platform capability.

## Details
Transaction Monitoring applies threshold, velocity, structuring, and peer-group scenarios alongside machine-learning models to flag suspicious activity for investigation and suspicious activity report filing. Alert scoring, network analytics, and feedback loops tune detection over time, while explainability and audit trails satisfy regulator and model-risk expectations.
