# Control Testing Execution

**Definition.** Control Testing Execution designs and runs sample-based and continuous control tests to evidence compliance-control effectiveness, delivering the BSA independent-testing pillar.
**Also known as:** Compliance Control Testing, Assurance Testing.

## Relationships
- Control Testing Execution is defined in the Compliance & Financial Crime domain.
- Control Testing Execution is derived from Compliance Monitoring & Testing.
- Control Testing Execution depends on the Analytics Platform capability.
- Control Testing Execution depends on the Governance Risk & Compliance Platform capability.

## Details
The BSA independent-testing pillar and the OFAC Framework testing-and-auditing component require objective testing of whether controls operate as designed. Control Testing Execution selects risk-based samples (e.g. CTR completeness and 15-day timeliness, SAR 30-day filing, screening match disposition quality, CDD/EDD file completeness), executes the test procedures, and rates each control's effectiveness. Continuous monitoring complements periodic sampling for high-volume controls. Exceptions are logged into issue and remediation tracking with severity and root cause.
