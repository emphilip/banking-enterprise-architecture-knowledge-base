# Value-Dated Posting

**Definition.** Value-Dated Posting applies effective and value dates to entries for correct interest and availability treatment, supporting the entry-fulfilment behaviour of the BIAN Current Account service domain.
**Also known as:** Value Dating, Effective-Date Posting.

## Relationships
- Value-Dated Posting is defined in the Deposits & Accounts domain.
- Value-Dated Posting is derived from Transaction Posting.
- Value-Dated Posting depends on the Transaction Posting Engine capability.

## Details
The BIAN Current Account entry-fulfilment behaviour assigns value dates; the value date sets when a deposit starts earning interest and, per Regulation CC (12 CFR 229) availability schedules, when the funds become withdrawable.
