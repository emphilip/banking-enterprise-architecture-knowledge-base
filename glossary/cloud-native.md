# Cloud-Native

**Definition.** Cloud-Native is an approach to building and running applications that exploits cloud computing through containers, microservices, declarative APIs, and elastic, resilient infrastructure.
**Also known as:** Cloud Native.

## Relationships
- Cloud-Native is related to the Technology Capability concept.
- Cloud-Native is related to the Core Banking concept.
- Cloud-Native mentions the Core Processing domain.

## Details
Cloud-native architectures, defined by the Cloud Native Computing Foundation, emphasize containers, orchestration (Kubernetes), and microservices. In banking, cloud-native cores and platforms (e.g. Thought Machine Vault, Mambu, Snowflake) supersede monolithic legacy systems, enabling scalability and faster delivery.
