# Core Banking

**Definition.** Core Banking is the back-end system of record that processes daily banking transactions, maintains account balances, and posts to the ledger across deposits, loans, and payments.
**Also known as:** Core Banking System.

## Relationships
- Core Banking is related to the Core Banking Processing capability.
- Core Banking is related to the Core Processing domain.
- Core Banking is related to the Cloud-Native concept.

## Details
Traditional core banking systems ran on mainframes (e.g. COBOL cores, Temenos T24), while modern cloud-native cores such as Thought Machine Vault, Mambu, and 10x Banking offer real-time, API-first, and configurable product engines. Core banking is the foundational system of record for retail and commercial banks.
