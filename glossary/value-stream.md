# Value Stream

**Definition.** Value Stream is a business architecture concept describing the end-to-end set of value-adding activities that create an outcome of value for a customer or stakeholder.
**Also known as:** Value Streams.

## Relationships
- Value Stream is related to the Business Capability concept.
- Value Stream is related to the BIAN concept.
- Value Stream mentions the APQC PCF concept.

## Details
A value stream is decomposed into stages, each enabled by business capabilities, and is a core artifact in frameworks such as TOGAF and the Business Architecture Guild's BIZBOK. In banking, value streams like customer onboarding or payment execution organize capability and process mapping.
